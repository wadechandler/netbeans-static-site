Instrucciones de Instalación de NetBeans IDE 6.1

Última actualización: August 9, 2012

Este documento describe como instalar NetBeans IDE 6.1 en su sistema. Para información acerca de los sistemas operativos soportados y configuraciones de hardware para el IDE, vea las Notas de Lanzamiento NetBeans IDE 6.1 . Vea Información NetBeans IDE 6.1 para aprender acerca de las nuevas características incluídas en NetBeans IDE 6.1.

Requerimientos de Software

Antes de instalar el IDE, el Kid de Desarrollo Java SE (JDK) 5 Actualización 14 (versión 1.5.0_14) o posterior (incluyendo JDK 6 Actualización 4) debe estar instalado en su sistema. Si usted no tiene una instalación de JDK 5 Actualización 14 o posterior, usted no puede continuar con la instalación. Para más información acerca de la versión requerida, vea la sección Requerimientos de Software en las Notas de Lanzamiento de NetBeans IDE 6.1 .

Opciones de Descarga del Instalador

En la Página de Descarga de NetBeans IDE 6.1, usted puede descargar uno de varios instaladores, cada uno contiene el IDE base y herramientas adicionales.

Los siguientes instaladores están disponibles:

  • Web & Java EE. Provee herramientas para desarrollar aplicaciones Java SE, Java EE y aplicaciones web. Esta opción de descarga también incluye el servidor GlassFish V2 UR2 y el software Apache Tomcat.
  • Mobility. Soporta el desarrollo de aplicaciones Java ME e incluye el paquete Mobility para ciertas aplicaciones MIDP para aparatos móbiles. Notese que algunas características de Mobility, tales como al asistente para Coneción Inalámbrica (Wireless Connection), requiere características que están disponibles en el paquete Web & Java EE. Si usted planea usar estas características, descargue la opción de instalación "All".
  • Java SE. Provee todas las características estándares para el desarrollo Java SE. El soporte para Desarrollo de Extensiones NetBeans (NetBeans Plugin Development) también está incluído.
  • Ruby. Provee herramientas para el desarrollo Ruby y soporta Rails y JRuby.
  • PHP. Provee herramientas para el desarrollo PHP. La opción de descarga PHP está disponible en Acceso Reciente (Early Access).
  • C/C++. Soporta el desarrollo en lenguaje C/C++.
  • All. Esta es la opción de descarga completa para NetBeans IDE 6.1. Contiene todos los paquetes y rutinas disponibles para el IDE. Incluye herramientas para el desarrollo de Arquitectura de Servicio Orientado (Service Oriented Architecture) y construcción de diagramas de Modelado Unificado de Lenguaje (Unified Modeling Language).

Personalizar Su Instalación

Si usted elige la opción de descarga Web & Java EE o "All", usted puede seleccionar exactamente cuales herramientas y rutinas desea instalar.

Para la opción de descarga Web & Java EE, en la página de bienvenida seleccione los servidores que quiera usar.

Para la opción de descarga "All", haga click en Personalizar (Customize) en la página de bienvenida y seleccione las características que quiera utilizar. Para más información, vea Instalar el Software.

Si quiere agregar funcionalidad a su configuración NetBeans IDE, use el Administrador de Extensiones (Plugin Manager) de NetBeans. Por ejemplo, asuma que usted comenzó descargando e instalando el paquete Ruby. Entonces decide que quiere probar la funcionalidad C/C++. Para hacer esto, vaya al Administrador de Extensiones en el IDE (seleccione Tools > Plugins) y agregue el paquete C/C++ a su instalación existente.

Existen varias notas importantes acerca de la instalación y configuración de algunas características de NetBeans IDE:

  1. Apache Tomcat 6.0.16 está incluído en las opciones de descarga "Web & Java EE" y "All" pero está predeterminado solamente el la opción "Web & Java EE". Para instalar Apache Tomcat desde la opción de descarga "All", ejecute el instalador y seleccione Apache Tomcat 6.0.16 en el cuadro de diálogo de Instalación Personalizada (Customize Installation).
  2. Para el desarrollo C/C++ (aplicable solo para las opciones de descarga C/C++ y "All"), usted necesita instalar adicionalmente los compiladores y las herramientas. Para mayor información, refiérase al Soporte de Instalación y Configuración C/C++.
  3. Si usted quiere usar el Administrador de Acceso del Sistema Sun Java (Sun Java System Access Manager), además de la opción de descarga "All" para IDE, usted necesita instalar la Plataforma de Aplicación Java SDK Actualización 4 o posterior desde http://java.sun.com/javaee/downloads/index.jsp.
  4. Para desarrollar aplicaciones Mobility para aparatos CDC usted necesita descargar e instalar Sun Java Toolkit 1.0 para CDC. Después de la instalación usted debe registrar la plataforma CDC en el IDE.
  5. Para Mac OS X, usted no puede personalizar la instalación de NetBeans IDE.

Comenzar la Descarga

  1. Navegue en la Página de Descarga de NetBeans IDE 6.1.
  2. En el área superior derecha de la página, seleccione su plataforma desde la lista desplegable.
  3. Haga click en el botón Download para las opciones de descarga que usted quiera instalar.
  4. Guarde el archivo de instalación en su sistema.

Instalar el Software

Siga las instrucciones en esta sección para instalar el IDE en su sistema. Estas instrucciones de instalación se aplican a todas las plataformas soportadas. Para una lista de plataformas soportadas, vea las Notas de Lanzamiento NetBeans IDE 6.1.

Microsoft Windows, Solaris OS y Linux

Para instalar el software:

  1. Después de completarse la descarga, ejecute el instalador.
    • Para Windows, el archivo ejecutable de instalación tiene extensión .exe. Haga doble click en el archivo de instalación para ejecutarlo.
    • Para plataformas Solaris y Linux, el archivo de instalación tiene extensión .sh. Para estas plataformas, usted necesita convertir los archivos de instalación en archivos ejecutables usando el siguiente comando: chmod +x ./<nombre-de-archivo-instalador>
  2. Para seleccionar las herramientas y rutinas a instalar, realice los siguientes pasos en la página de Bienvenida (Welcome) del asistente de instalación:
    1. Haga click en Personalizar (Customize).
    2. En el cuadro de diálogo Personalizar, haga sus selecciones.
    3. Haga click en OK.
  3. en la página de Bienvenida (Welcome) del asistente de instalación, haga click en Next.
  4. En la página de Acuerdo de Licencia (License Agreement), revise el acuerdo de licencia, chequee la casilla de aceptación y haga click en Next.
  5. En la página de instalación de NetBeans IDE, haga lo siguiente:
    1. Acepte el directorio de instalación predeterminado para NetBeans IDE o especifique otro directorio.
      Nota: El directorio de instalación debe estar vacío, y el perfil de usuario que usted está usando debe tener permiso de lectura/escritura para este directorio.
    2. Acepte la instalación predeterminada JDK para utilizar con NetBeans IDE o seleccione una instalación diferente desde la lista desplegable.
      Si el asistente de instalación no halla una instalación JDK compatible para su uso con NetBeans IDE, su JDK no se encuentra instalado en el directorio por defecto. En ese caso, especifique el directorio de instalación JDK o cancele la instalación en curso e instale la versión JDK necesaria y reinicie la instalación de NetBeans IDE.
    3. Haga click en Next.
  6. Si se abre la página de instalación del servidor GlassFish, haga lo siguiente:
    1. Acepte el directorio de instalación predeterminado para el servidor o especifique otro directorio de instalación.
      Nota: El directorio de instalación que usted especifique debe estar vacío, y el perfil de usuario que usted está usando debe tener permiso de lectura/escritura para este directorio.
    2. Desde la lista de instalaciones compatibles JDK, elija la JDK que quiera que su servidor utilice o haga click en el botón Explorar (Browse) para especificar otra instalación JDK.
    3. Cambie el nombre de usuario y contraseña para el dominio del servidor o acepte los que aparecen por defecto y haga click en Next.
      El nombre de usuario y la contraseña que aparecen por defecto son admin y adminadmin respectivamente.
    4. Verifique los valores predeterminados de los puertos (HTTP, HTTPS y Admin) para el servidor o cambielos si es necesario.
    5. Haga click en Next.
  7. Si se abre la página de instalación de la aplicación Apache Tomcat, acepte el directorio de instalación predeterminado o especifique otro directorio de instalación.
  8. En la página Resumen (Summary), verifique que la lista de componentes a ser instalados es correcta y que usted tienen espacio suficiente en su sistema para la instalación.
  9. Haga click en Install para comenzar la instalación.
  10. Cuando se complete la intalación:
    1. Si la aplicación GlassFish ha sido instalada, para registrarla seleccione la casilla de verificación Register the Sun Java System Application Server and NetBeans IDE después de finalizar la instalación.
    2. Si la aplicación GlassFish no ha sido instalada, seleccione la casilla de verificación Register the NetBeans IDE después de finalizar la instalación para registrar su NetBeans IDE.
    3. Haga click en Finish para salir del asistente.
  11. Si usted seleccionó la casilla de verificación de Registro en el paso previo, la página de Registro se abre en el navegador predeterminado. Haga click en Registrar (Register).
  12. Introduzca su Cuenta En Línea Sun o cree una en la página www.sun.com, la cual está accesible via link.

Nota: Si encuentra problemas para completar satisfactoriamente la instalación, vea Problemas para ver descripciones y soluciones sugeridas para problemas no resueltos que puedan afectar el proceso de instalación.

Nota: Si usted no ha registrado la aplicación GlassFish después de la instalación, puede hacerlo cuando ejecute NetBeans IDE o cuando abra la Consola de Administración del servidor GlassFish.

Nota: Si usted no ha registrado su NetBeans IDE 6.1 justo después de la instalación, puede hacerlo cuando ejecute NetBeans IDE.

Mac OS X

Para instalar el software:

  1. Después de completar la descarga, ejecute el instalador. El archivo de instalación tiene la extensión .dmg.
  2. En el panel que se abre haga click en el ícono del paquete. El paquete tiene la extensión .mpkg. El asistente de instalación comienza.
  3. En la página de Bienvenida (Welcome) del asistente, haga click en Continuar (Continue).
  4. Revise el acuerdo de licencia y haga click en Continue. Haga click en Aceptar (Accept) en la ventana emergente para aceptar la licencia.
  5. En la página de Selección de Destino (Select a Destination), seleccione el disco y haga click en Continue.
  6. Si usted descargó los agregados All o Java EE, usted puede personalizar su instalación. En el último panel de instalación del asistente, presione Personalizar (Customize) en la esquina inferior izquierda del panel. El arbol de productos es mostrado.
  7. Seleccione los productos que quiera instalar
  8. Introduzca el nombre y contraseña del administrador de su sistema y haga click en OK para comenzar la instalación.

Desinstalar el Software

Microsoft Windows

Si es necesario, usted puede desinstalar NetBeans IDE, el servidor GlassFish y la aplicación Apache Tomcat usando los siguientes procedimientos.

Para desinstalar el IDE:

  1. Cierre el IDE.
  2. Desde el Panel de Control, selecciones Agregar o Quitar Programas, después seleccione NetBeans IDE 6.1 y haga click en Quitar.

Para desinstalar el servidor GlassFish:

  1. Cierre el IDE.
  2. Desde el Panel de Control, selecciones Agregar o Quitar Programas, después seleccione GlassFish V2 UR2 y haga click en Quitar.

Para desinstalar Apache Tomcat:

  1. Cierre el IDE.
  2. Desde el Panel de Control, selecciones Agregar o Quitar Programas, después seleccione Apache Tomcat 6.0.16 y haga click en Quitar.

Solaris OS, Linux

Si es necesario, usted puede desinstalar NetBeans IDE, el servidor GlassFish y la aplicación Apache Tomcat usando los siguientes procedimientos.

Para desinstalar el IDE:

  1. Cierre el IDE.
  2. En el directorio de instalación del IDE, ejecute el archivo uninstall.sh.
  3. Para desinstalar GlassFish/Tomcat junto con el IDE en la página Summary (Resumen) asegúrese que las versiones correctas están listadas y que está seleccionadas las casillas de verificación apropiadas.
  4. En la página Summary (Resumen), haga click en Uninstall (Desinstalar).
  5. Luego de completada la instalación, haga click en Finish (Finalizar).

Para desinstalar el servidor GlassFish:

  1. Cierre el IDE.
  2. Ubique el directorio de instalación del servidor.

    Nota: La ubicación de instalación predeterminada es diferente dependiendo del tipo de instalación y de la plataforma. La siguiente tabla muestra los directorios de instalación predeterminados del servidor.

    Plataforma Instalado Como... Instalación SOA
    Solaris OS 
    		 root /opt/glassfish-v2ur2
    Solaris OS,
    Linux    		 user ~/glassfish-v2ur2
    Linux root /usr/local/glassfish-v2ur2
  3. En el directorio de instalación del servidor, ejecute el archivo uninstall.sh.
  4. En la página Summary (Resumen), haga click en Uninstall (Desinstalar).
  5. Despues de terminar la desinstalación, haga click en Finish (Finalizar).

Nota: El servidor GlassFish es requerido para la funcionalidad SOA. Cuando desinstale el servidor, la funcionalidad SOA es removida automáticamente de la instalación de NetBeans IDE.

Para desinstalar Apache Tomcat:

  1. Cierre el IDE.
  2. Ubique el directorio de instalación de Apache Tomcat y ejecute el archivo uninstall.sh.
  3. En la página Summary (Resumen), haga click en Uninstall (Desinstalar).
  4. Despues de terminar la desinstalación, haga click en Finish (Finalizar).

Mac OS X

Para desinstalar el IDE y el servidor:

  1. Cierre el programa.
  2. Arrastre las carpetas del programa o los archivos del programa a la Papelera.
    Note que la ubicación predeterminada para el IDE y el servidor es /Applications/NetBeans/.

Actualizar la Versión y Coexistir con Versiones Anteriores

NetBeans IDE 5.5 o 5.5.1, o 6.0 pueden coexistir con NetBeans IDE 6.1 en el mismo sistema. Usted no tendrá que desinstalar las versiones 5.5 o 5.5.1, 6.1 para poder instalar o ejecutar NetBeans IDE 6.1.

Si posee NetBeans IDE 5.5, 5.5.1, o 6.0, durante la primera ejecución de NetBeans IDE 6.1, usted podrá elegir si importar las configuraciones de usuario desde los directorios de usuario anteriores.

Problemas

Acerca del Directorio de Trabajo del Instalador

El instalador almacena datos acerca de las sesiones de instalación en el directorio de trabajo del instalador. Ejemplos de estos datos son la lista de componentes instalados en sus sistema, el registro de mensajes de instalación y datos de registro.

La ubicación predeterminada del directorio de trabajo del instalador es %PERFILDEUSUARIO%\.nbi en Windows y ~/.nbi en otras plataformas. El subdirectorio log contiene los archivos de registro de la instalación.

Limpiar Luego de Instalaciones Incompletas

Después de instalaciones incompletas o inexitosas, posiblemente necesite limpiar su sistema de instalaciones previas.

Para Microsoft Windows, Solaris OS y Linux:

  1. Elimine el directorio de trabajo del instalador.
  2. Elimine los directorios de instalación del IDE y del servidor GlassFish.
  3. En la plataforma Windows, borre todas las entradas de registro nbi* en las siguientes claves de registro:
    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall
    HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall

Para Mac OS X:

  • Arrastre las carpetas o los archivos del programa a la Papelera.

Deshabilite Algunos Software Antes de Instalar

Es mejor desabilitar los cortafuegos, antivirus y monitores de aplicaciones antes de comenzar su instalación. Algunas de estas aplicaciones pueden evitar que los componentes IDE sean configurados correctamente durante el proceso de instalación.

Por ejemplo, si usted tiene la aplicación Logitech Process Monitor (LVPrcSrv.exe) ejecutándose en su sistema, ciérrela antes de la instalación.
