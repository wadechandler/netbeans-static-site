Информация о выпуске среды IDE NetBeans 7.1

IDE NetBeans представляет собой отмеченную наградами среду разработки, доступную для ОС Windows, Mac, Linux и Solaris. Проект NetBeans состоит из среды IDE с открытым исходным кодом и платформы приложения, которая позволяет разработчикам быстро создавать веб-, корпоративные, настольные и мобильные приложения с помощью платформы Java, а также PHP, JavaScript и Ajax, Groovy и Grails, а также C/C++.

Проект NetBeans поддерживается активным сообществом разработчиков, которое предлагает обширную документацию и обучающие материалы, а также множество сторонних подключаемых модулей.

Обзор выпуска

Теперь IDE NetBeans 7.1 предоставляет поддержку JavaFX 2.0, обеспечивая при этом полный цикл разработки с функциями компиляции, отладки и профилирования приложений JavaFX 2.0. В этот выпуск также добавлены значительные усовершенствования Swing GUI Builder, поддержку CSS3 и средства визуальной отладки пользовательских интерфейсов Swing и JavaFX. В число дополнительных возможностей входит встроенная в среду IDE поддержка Git, новые функции отладки PHP, различные усовершенствованные возможности JavaEE и Maven и многое другое.

Среда IDE NetBeans 7.1 поддерживает английский, бразильский португальский, русский, упрощенный китайский и японский языки.

Получение IDE NetBeans 7.1


Чтобы использовать поддержку JavaFX 2.0, загрузите пакет SDK JavaFX 2.0 с веб-сайта JavaFX.com

Обзор новых функций

JavaFX

  • Поддержка JavaFX 2.0
  • Поддержка всех трех моделей развертывания: рабочий стол, апплет, JNLP
  • Средства предварительной загрузки для оптимизации процесса загрузки приложения
  • Настроенные элементы управления интерфейсом пользователя с использованием CSS3

Java

  • Новый визуальный отладчик
  • Поддержка промежутков в настройщике GridBagLayout
  • Анализ и реорганизация кода
  • Улучшенные выделение и форматирование Javadoc

Веб-языки

  • Поддержка CSS3
  • Автозавершение кода и документация для новых элементов CSS3
  • Свойства, связанные с обозреваателем

Java EE

  • Поддержка развертывания кластеров и экземпляров для GlassFish
  • Разработка приложений Java EE 6 для развертывания на сервере WebLogic
  • Более 50 усовершенствований CDI в редакторе Java
  • Поддержка набора компонентов JSF
  • Усовершенствования в Java, веб-службах, EJB, WebLogic и др.

PHP

  • Улучшенная отладка PHP
  • Возможности групп тестов PHPUnit
  • Поддержка шаблонов Smarty
  • Более быстрая загрузка с поддержкой активности клиента (S)FTP


Платформа NetBeans

  • Новые интерфейсы API для оконной системы
  • Усовершенствованное поведение TopComponentGroup
  • Визуальный конструктор компоновки окон
  • Подключаемые компоненты с несколькими представлениями

Управление версиями

  • Встроенная поддержка Git
  • Вкладка истории в окне редактора файлов
  • Блокирование и разблокирование файлов в рабочем каталоге Subversion
  • Поддержка ветвей и тегов Mercurial
