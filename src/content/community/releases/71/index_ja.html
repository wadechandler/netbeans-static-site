NetBeans IDE 7.1 リリース情報

NetBeans IDE は、受賞歴のある統合開発環境で、Windows、Mac、Linux、および Solaris で使用可能です。NetBeans プロジェクトは、オープンソースの IDE とアプリケーションプラットフォームから構成され、これを使用すると、開発者は Java プラットフォームのほか、PHP、JavaScript と Ajax、Groovy と Grails、および C/C++ を使用して、Web アプリケーション、エンタープライズアプリケーション、デスクトップアプリケーション、およびモバイルアプリケーションを短時間で作成できます。

NetBeansプロジェクトは活発な開発者コミュニティーによってサポートされ、豊富なドキュメントとトレーニングリソース、およびさまざまなサードパーティーのプラグイン提供します。

リリース概要

NetBeans IDE 7.1 は、JavaFX 2.0 アプリケーションのコンパイル/デバッグ/プロファイルの完全な開発サイクルを可能にすることによって、JavaFX 2.0 のサポートを導入しています。このリリースは、重要な Swing GUI ビルダー拡張機能、CSS3 のサポート、Swing および JavaFX ユーザーインタフェースのビジュアルデバッグ用のツールも提供します。その他の重要点には、IDE に組み込まれた Git サポート、新しい PHP デバッグ機能、JavaEE および Maven のさまざまな改善などがあります。

NetBeans IDE 7.1 は、英語、ポルトガル語 (ブラジル)、日本語、ロシア語、および簡体字中国語で使用できます。

NetBeans IDE 7.1 の取得


JavaFX 2.0 サポートを使用するには、JavaFX.com から JavaFX 2.0 SDK をダウンロードします

新機能の概要

JavaFX

  • JavaFX 2.0 のサポート
  • 3 つすべての配備モデルのサポート: デスクトップ、アプレット、JNLP
  • アプリケーションの読み込みの操作性を向上させるプレローダー
  • CSS3 を使用したカスタマイズされた UI コントロール

Java

  • 新しいビジュアルデバッガ
  • GridBagLayout カスタマイザのギャップのサポート
  • 検査とリファクタリング
  • Javadoc の強調表示と書式設定の向上

Web 言語

  • CSS3 のサポート
  • コード補完と新しい CSS3 要素のドキュメント
  • ブラウザ固有のプロパティー

Java EE

  • GlassFish のクラスタおよびインスタンス配備のサポート
  • WebLogic Server に配備する Java EE 6 アプリケーションの開発
  • Java エディタの 50 以上の CDI 拡張機能
  • JSF コンポーネントスイートのサポート
  • Java Persistence、Web サービス、EJB、WebLogic などの改善

PHP

  • PHP デバッグの向上
  • PHPUnit テストグループ機能
  • Smarty テンプレートのサポート
  • (S)FTP クライアントのキープアライブによるアップロードの高速化

NetBeans プラットフォーム

  • 新しいウィンドウシステム API
  • TopComponentGroup 動作の拡張
  • ビジュアルウィンドウレイアウトデザイナ
  • プラグイン可能なマルチビューコンポーネント

バージョン管理

  • 統合 Git サポート
  • ファイルエディタウィンドウの「履歴」タブ
  • Subversion 作業ディレクトリ内のファイルのロック/ロック解除
  • Mercurial 分岐およびタグのサポート

このリリースの機能の詳細は、NetBeans IDE 7.1 の New and Noteworthy ページを参照してください。
