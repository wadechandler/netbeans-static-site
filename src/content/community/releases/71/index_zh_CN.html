NetBeans IDE 7.1 发行信息

NetBeans IDE 是一个荣获奖项的集成开发环境，可用于 Windows、Mac、Linux 和 Solaris。NetBeans 项目由开源 IDE 和应用程序平台组成；通过该 IDE 和平台，开发者可以使用 Java 平台以及 PHP、JavaScript 和 Ajax、Groovy 和 Grails、C/C++ 快速创建 Web、企业、桌面和 Mobile 应用程序。

NetBeans 项目由活跃的开发者社区支持，并且还提供了广泛的文档和培训资源，以及各种供您选用的第三方插件。

发行版概述

NetBeans IDE 7.1 通过实现 JavaFX 2.0 应用程序的完整编译/调试/分析开发周期，引入了对 JavaFX 2.0 的支持。该发行版本还显著改进了 Swing GUI 生成器，并且提供了 CSS3 支持，以及用于可视调试 Swing 和 JavaFX 用户界面的工具。其他重要功能包括集成到 IDE 中的 Git 支持、新的 PHP 调试功能，以及各种 JavaEE 和 Maven 改进功能等。

NetBeans IDE 7.1 提供了英语、巴西葡萄牙语、日语、俄语和简体中文版本。

获取 NetBeans IDE 7.1


要使用 JavaFX 2.0 支持，请从 JavaFX.com 下载 JavaFX 2.0 SDK

新增功能集锦

JavaFX

  • 支持 JavaFX 2.0
  • 支持所有三种部署模型：桌面、Applet 和 JNLP
  • 改善应用程序装入体验的预加载器
  • 使用 CSS3 的定制 UI 控件

Java

  • 新的可视调试器
  • GridBagLayout 定制器中的间距支持
  • 检查和重构
  • 改进的 Javadoc 突出显示和格式设置

Web 语言

  • 支持 CSS3
  • 用于新的 CSS3 元素的代码完成和文档生成
  • 特定于浏览器的属性

Java EE

  • GlassFish 的群集和实例部署支持
  • 开发 Java EE 6 应用程序以部署到 WebLogic Server
  • Java 编辑器中的 50 多项 CDI 改进功能
  • JSF 组件套件支持
  • 改进了 Java 持久性、Web 服务、EJB 和 WebLogic 等

PHP

  • 改进了 PHP 调试功能
  • PHPUnit 测试组功能
  • 支持 Smarty 模板
  • 通过“维持连接”功能加快 (S)FTP 客户端的上载速度


NetBeans 平台

  • 新的窗口系统 API
  • 改进了 TopComponentGroup 行为
  • 可视窗口布局设计器
  • 可插接的多视图组件

版本控制

  • 一体式 Git 支持
  • “文件编辑器”窗口中的“历史记录”标签
  • 锁定/解锁 Subversion 工作目录中的文件
  • 支持 Mercurial 分支和标记

有关此发行版本功能的详细信息，请参见 NetBeans IDE 7.1 新增功能和重要功能页。
