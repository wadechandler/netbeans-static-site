Указания по установке среды IDE NetBeans 6.9

Дата последнего обновления: 11 июня 2010 г.

В этом документе описывается процедура установки среды IDE NetBeans 6.9. Для получения сведений о поддерживаемых операционных системах и комплектации оборудования для среды IDE обратитесь к "Заметкам к выпуску среды IDE NetBeans 6.9". Для получения сведений о новых функциях, вошедших в данный выпуск среды IDE, обратитесь к странице Информация к выпуску среды IDE NetBeans 6.9.

Требуемое программное обеспечение

Для установки среды IDE NetBeans требуется пакет разработчика Java SE (JDK) 6.0, обновление 13 или выше. Среду IDE версии 6.9 невозможно установить при использовании JDK 5.0. Новейшую версию обновления JDK 6.0 можно загрузить по адресу http://java.sun.com/javase/downloads.

Для установки и запуска пакетов Netbeans для PHP, Ruby и C/C++ требуется только установка среды Java Runtime Environment (JRE) 6. Однако если планируется использовать любую из функций Java, потребуется JDK 6.0.

в начало

Варианты загрузки программы установки

На странице загрузки среды IDE NetBeans можно загрузить одну из нескольких программ установки, каждая из которых содержит основные и дополнительные средства IDE.

Доступны следующие программы установки:

  • Java SE. Поддерживает все стандартные функции разработки Java SE, а также платформу разработки NetBeans RCP.
  • JavaFX. Включает в себя пакет JavaFX SDK 1.3 и JavaFX Composer для разработки многоплатформенных приложений с широкими возможностями.
  • Java. Предоставляет средства для разработки приложений Java SE, Java EE и Java ME, а также поддержку платформы разработки RCP NetBeans. Этот вариант загрузки также включает в себя сервер GlassFish Server Open Source Edition 3.0.1 и программное обеспечение Apache Tomcat 6.0.26.
  • Ruby. Предоставляет средства для разработки приложений Ruby и включает в себя поддержку Rails и JRuby. Также включает в себя сервер GlassFish Server Open Source Edition 3.0.1.
  • C/C++. Реализована поддержка разработки на языках C, C++, Qt, Фортран и на языке ассемблера.
  • PHP. Предоставляет средства для разработки PHP 5.x и поддержки платформ Zend и Symfony.
  • Все. Это вариант полной загрузки, содержащий все среды выполнения и технологии, доступные для среды IDE NetBeans.

в начало

Индивидуализация установки

При выборе вариантов загрузки "Java" или "Все" можно точно указать средства и среды выполнения, которые требуется установить. Щелкните ссылку "Настроить" на странице "Добро пожаловать" и выберите требуемые компоненты. Для получения дополнительных сведений обратитесь к разделу Установка программного обеспечения.

Для добавления функциональных возможностей к среде IDE NetBeans используется диспетчер подключаемых модулей NetBeans. Например, предположим, что сначала был загружен и установлен пакет "Ruby". После этого было принято решение опробовать функциональность C/C++. Для этого в среде IDE перейдите к диспетчеру подключаемых модулей NetBeans (выберите "Сервис > Подключаемые модули") и добавьте к существующей установке пакет "C/C++".

Ниже приводится ряд важных примечаний относительно установки и настройки некоторых компонентов среды IDE NetBeans:

  1. Пакет Apache Tomcat 6.0.26 включен в варианты загрузки "Java" и "Все", но по умолчанию не устанавливается ни в одном из этих вариантов. Для установки Apache Tomcat из пакета "Java" или "Все" запустите программу установки и выберите "Apache Tomcat 6.0.26" в диалоговом окне "Изменить устанавливаемые компоненты".
  2. Для разработки на C/C++ (применимо для вариантов загрузки "C/C++" и "Все") необходимо установить компиляторы и средства. Для получения подробных сведений обратитесь к документу Настройка среды IDE NetBeans для C/C++/Фортрана.
  3. Для платформы Mac OS X можно настроить установку среды IDE NetBeans из вариантов загрузки "Java", "Ruby" и "Все". Установку среды IDE из пакетов загрузки для JavaSE, PHP и C/C++ настроить нельзя.
  4. Также можно загрузить независимый от платформы zip-файл и запустить исполняемый файл среды IDE NetBeans. Следует отметить, что GlassFish Server Open Source Edition и Apache Tomcat доступны только с программами установки для конкретных платформ.

Начало загрузки

  1. Перейдите по адресу https://netbeans.org/downloads/
  2. Выберите соответствующую платформу и язык из выпадающего списка в верхней правой части страницы. Также можно выбрать для загрузки и использования zip-файл, независимый от платформы.
  3. Нажмите кнопку "Загрузить", соответствующую выбранному варианту загрузки для установки.
  4. Сохраните файл программы установки.

в начало

Установка программ

Для установки среды IDE следуйте указаниям, приведенным в этом разделе. Данные указания по установке применимы ко всем поддерживаемым платформам. Список поддерживаемых платформ приводится в заметках к выпуску.

ОС Microsoft Windows, Solaris и Linux

Для установки программ:

  1. После завершения загрузки запустите программу установки.
    • Для Windows исполняемый файл программы установки имеет расширение .exe. Чтобы запустить файл программы установки, дважды щелкните его.
    • Для платформ Solaris и Linux файл программы установки имеет расширение .sh. Для этих платформ необходимо выполнить следующую команду для исполняемых файлов программы установки: chmod +x <installer-file-name>
  2. Для выбора средств и среды исполнения выполните следующие действия на странице "Добро пожаловать" мастера установки:
    1. Нажмите кнопку "Настроить".
    2. В диалоговом окне "Изменить устанавливаемые компоненты" выберите требуемые варианты.
    3. Нажмите кнопку "ОК".
  3. На странице "Добро пожаловать" мастера установки нажмите кнопку "Далее".
  4. На странице "Лицензионное соглашение" ознакомьтесь с лицензионным соглашением, установите флажок для принятия его условий и нажмите кнопку "Далее".
  5. На странице установки среды IDE NetBeans выполните следующие действия:
    1. Выберите каталог для установки среды IDE NetBeans по умолчанию или укажите другой каталог.

      Примечание. Каталог установки должен быть пустым, а профиль пользователя, в котором выполняется программа установки, должен предусматривать полномочия на чтение и запись для этого каталога.

    2. Выберите установку комплекта JDK для использования со средой IDE NetBeans по умолчанию или укажите другую установку из раскрывающегося списка. Если мастеру установки не удается обнаружить совместимую установку комплекта JDK для использования со средой IDE NetBeans, значит, комплект JDK не установлен в местоположении по умолчанию. В этом случае укажите путь к установленному пакету JDK и нажмите кнопку "Далее" или отмените текущую установку. После установки необходимой версии пакета JDK можно перезапустить установку.
  6. При появлении страницы установки GlassFish Server Open Source Edition 3.0.1 выберите каталог установки по умолчанию или укажите другое местоположение для установки.
  7. Если выполняется установка сервера приложений Apache Tomcat, выберите на его странице установки каталог установки по умолчанию или укажите другое местоположение для установки.
  8. На странице "Сводка" проверьте правильность списка компонентов, которые требуется установить, а также наличие достаточного пространства в системе для установки.
  9. Для запуска процесса установки нажмите кнопку "Установить".

Примечание. При возникновении проблем во время установки пакета обратитесь к разделу Устранение неполадок, в котором содержится описание соответствующих процедур и возможных обходов проблем, влияющих на процесс установки.

Mac OS X

Для установки программ:

  1. После завершения загрузки запустите программу установки. Файл программы установки имеет расширение .dmg.
  2. На открывшейся панели щелкните значок пакета. Пакет имеет расширение .mpkg. Запускается мастер установки.
  3. На странице "Добро пожаловать" мастера установки нажмите кнопку "Продолжить".
  4. Прочтите лицензионное соглашение и нажмите кнопку "Продолжить". Для принятия условий лицензионного соглашения нажмите кнопку "Принимаю" во всплывающем окне.
  5. На странице "Выберите место назначения" выберите диск и нажмите кнопку "Продолжить".
  6. Если были загружены пакеты "Все", "Java" или Ruby, можно настроить индивидуальную установку. В нижней левой части последней панели мастера установки нажмите кнопку "Настроить". Появится дерево программ.
  7. Выберите программы, которые требуется установить.
  8. Введите соответствующее имя администратора и пароль и нажмите кнопку "ОК" для запуска установки.

Пакет, независимый от платформы

Запуск программного обеспечения:

  1. После загрузки независимого от платформы zip-файла распакуйте его в любую папку на компьютере.
  2. Запустите исполняемый файл, расположенный в каталоге netbeans/bin.
  3. Примите условия лицензионного соглашения. Будет запущена среда IDE NetBeans.

Примечание. Если совместимый установленный комплект JDK не обнаруживается, может потребоваться вручную добавить путь к нему в папку установки JDK, выполнив следующие действия:

  1. Откройте в текстовом редакторе файл netbeans.conf, расположенный в каталоге netbeans/etc.
  2. Введите местоположение совместимого установленного комплекта JDK в качестве значения параметра netbeans_jdkhome. В Windows расположение по умолчанию – C:\Program Files\Java\jdk1.6.0.
  3. Сохраните файл netbeans.conf и запустите исполняемый файл в каталоге netbeans/bin.

Примечание. GlassFish Server Open Source Edition и Apache Tomcat доступны только с программами установки для конкретных платформ.

Удаление программ

Для удаления среды IDE следуйте указаниям, приведенным в этом разделе. Данные указания по удалению применимы ко всем поддерживаемым платформам. Список поддерживаемых платформ приводится в заметках к выпуску.

Microsoft Windows

При необходимости среду IDE NetBeans, GlassFish Server Open Source Edition 3.0.1 и Apache Tomcat можно удалить путем выполнения следующей последовательности действий.

Удаление среды IDE:

  1. Завершите работу среды IDE.
  2. Выберите в "Панели управления" пункт "Установка и удаление программ", после чего выберите среду IDE NetBeans 6.9 и нажмите кнопку "Удалить".

Для удаления GlassFish Server Open Source Edition 3.0.1:

  1. Завершите работу среды IDE.
  2. Выберите в "Панели управления" пункт "Установка и удаление программ", после чего выберите GlassFish Server Open Source Edition 3.0.1 и нажмите кнопку "Удалить".

Удаление пакета Apache Tomcat:

  1. Завершите работу среды IDE.
  2. Выберите в "Панели управления" пункт "Установка и удаление программ", после чего выберите Apache Tomcat 6.0.26 и нажмите кнопку "Удалить".

ОС Solaris и Linux

При необходимости среду IDE NetBeans, GlassFish Server Open Source Edition 3.0.1 и Apache Tomcat можно удалить путем выполнения следующей последовательности действий.

Удаление среды IDE:

  1. Завершите работу среды IDE.
  2. В каталоге установки IDE запустите файл uninstall.sh.
  3. Для удаления GlassFish и Tomcat вместе со средой IDE на странице "Сводка" проверьте правильность версий и соответствующих флажков.
  4. На странице "Сводка" нажмите кнопку "Удалить".
  5. После завершения удаления нажмите кнопку "Готово".

Удаление GlassFish Server Open Source Edition 3.0.1:

  1. Завершите работу среды IDE.
  2. Определите каталог установки сервера приложений.

    Примечание. Местоположение установки по умолчанию зависит от типа установки и платформы. В следующей таблице приводятся каталоги установки сервера приложений по умолчанию для разных платформ.

    Платформа Установка от имени... Установка GlassFish 3.0.1
    Windows пользователь с правами администратора %ProgramFiles%\glassfish-3.0.1
    Windows user %USERPROFILE%\glassfish-3.0.1
    ОС Solaris root /opt/glassfish-3.0.1
    ОС Solaris,Linux user ~/glassfish-3.0.1
    Linux root /usr/local/glassfish-3.0.1
  3. В каталоге установки сервера приложений запустите файл uninstall.sh.
  4. На странице "Сводка" нажмите кнопку "Удалить".
  5. После завершения удаления нажмите кнопку "Готово".

Удаление пакета Apache Tomcat:

  1. Завершите работу среды IDE.
  2. Определите каталог установки пакета Apache Tomcat и запустите файл uninstall.sh.
  3. На странице "Сводка" нажмите кнопку "Удалить".
  4. После завершения удаления нажмите кнопку "Готово".

Mac OS X

Удаление среды IDE и сервера приложений:

  1. Завершите работу программного обеспечения.
  2. Перетащите папки или файлы программы в корзину. Учтите, что местоположением для среды IDE и сервера приложений по умолчанию является /Applications/NetBeans/.

Пакет, независимый от платформы

Для удаления файлов среды IDE выполните последовательность действий, приведенную в разделе этого документа Очистка после незавершенных установок.

Поддержка обновлений и множественных установок

Одновременно со средой IDE NetBeans 6.9 в одной системе могут существовать множественные установки среды IDE NetBeans 5.x и 6.x. Для установки или запуска среды IDE NetBeans 6.9 удаление предыдущих версий не требуется.

Если в системе уже установлена среда IDE NetBeans, то при первом запуске NetBeans IDE 6.9 можно выбрать, следует ли импортировать пользовательские настройки из существующего пользовательского каталога.

Примечание. На этом этапе невозможно выполнить последовательные обновления версий среды IDE. Для обновления до новейшей версии необходимо загрузить самую позднюю версию установщика.

Устранение неполадок

О рабочем каталоге программы установки

Программа установки сохраняет данные о сеансах установки в рабочем каталоге. Эти данные включают список установленных в системе компонентов, сообщения файла регистрации установки и информацию из реестра.

Местоположением рабочего каталога программы установки по умолчанию является %USERPROFILE%\.nbi для ОС Windows и ~/.nbi для других платформ. Файлы регистрации установки содержатся в подкаталоге файла регистрации.

Очистка после незавершенных установок

После незавершенных или неуспешных установок может потребоваться очистка системы.

Для ОС Microsoft Windows, Solaris и Linux:

  1. Удалите рабочий каталог программы установки.
  2. Удалите каталоги установки среды IDE и сервера приложений GlassFish.
  3. На платформе Windows следует удалить все записи вида nbi* из следующих ключей реестра: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall, HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall

Для Mac OS X:

  • Перетащите папки или файлы программы в корзину.

Отключение программного обеспечения перед установкой

Перед началом установки рекомендуется отключить брандмауэр, антивирусное программное обеспечение и приложения мониторинга. Некоторые из этих приложений могут стать причиной некорректной настройки компонентов среды IDE в течение процесса установки.

Например, если в системе выполняется приложение монитора процессов Logitech (LVPrcSrv.exe), перед установкой его рекомендуется отключить.
