NetBeans IDE 6.5 릴리즈 정보

NetBeans IDE는 Windows, Mac, Linux, and Solaris에서 사용할 수 있는 우수한 통합 개발환경환경 입니다. NetBeans 프로젝트는 오픈소스 IDE 와 개발자들이 자바 플랫폼을 이용하여 빠르게 웹, 기업, 데스크톱과 모바일 애플리케이션과 PHP, JavaScript and Ajax, Ruby and Ruby on Rails, Groovy, and C/C++ 를 생성 할 수 있는 애플리케이션 플랫폼를 포함하고 있습니다. 이 IDE는 활성한 개발 커뮤니티에 의해 지원되고 있고 제3자의 여러 플러그인들을 제공하고 있습니다.

릴리즈 개요

자바 플렛폼 (Java SE, Java EE, Java ME, and JavaFX)의 완변한 지원뿐 아니라 PHP, Ajax and JavaScript, Groovy and Grails, Ruby and Ruby on Rails, and C/C++같은 소프트웨어 개발에도 NetBeans 6.5 IDE는 이상적인 도구입니다. 6.5 릴리즈는 웹 프레임워크(Hibernate, Spring, JSF, JPA), GlassFish 애플리케이션 서버와 데이터베이스의 향상된 지원을 제공합니다. IDE-전역 빠른검색 단축키, 사용하기 쉬운 인터페이스와 저장시 자동 컴파일을 포함합니다.

The NetBeans IDE 6.5.1 bug fix release is a minor update to NetBeans IDE 6.5.

NetBeans IDE 6.5 가지기

문서

NetBeans IDE 6.5를 시작하시려면 다음 문서를 사용하세요.

  • 6.5 랄라주 메모
  • 6.5 설치 설명

기능 하이라이트

PHP

  • PHP 에디터 (코드완성, syntactic and semantic code highlighting)
  • heredoc notation and PHTML 지원
  • Xdebug를 사용한 디버깅
  • MySQL 데이터베이스 코드 조각 생성

JavaScript and Ajax

  • FireFox & Internet Explorer에서 JavaScript 디버깅
  • HTTP 트랜젝션 모니터링
  • 향상된 JavaScript and CSS 에디터
  • Yahoo UI, Woodstock, jQuery, Dojo, Scriptaculous, Prototype libraries을 포함한 JavaScript 라이브러리 관리자

Java EE & 웹 개발

  • Spring, Hibernate, JSF, JSF CRUD 생성기, JPA (Java Persistence API)의 향상된 지원
  • 데이터베이스 테이블또는 JPA 엔티티에서 RESTful 웹서비스 생성
  • PHP 파일에 SaaS서비스 Drag and drop
  • SQL 에디터 향상 (코드완성, 쿼리 저장/소환, 등등)
  • 향상된 Eclipse 프로젝트 가져오기와 자바 웹 프로젝트 동기화
  • 저장시 자동 설치(Deploy)

JavaFX

  • JavaFX projects을 생성, 테스트, 디버그, 프로파일하고 설치(Deploy)
  • JavaFX 에디터와 더불어 문법 강조, 소스 둘러보그, 코드완성, 코드접기, javadoc 팝업과 오류 탐지
  • 시각 요소의 실시간 미리보기
  • 코드 조각 Code snippet palette for transformations, effects and animations

Groovy and Grails

  • 순수 Groovy 애플리케이션 개발하거나 자바 SE 프로젝트에서 Groovy이용
  • Groovy 에디터와저불어 코드완성, 문법강조 그리고 더 많은 기능들
  • Grails 웹 애플리케이션 프레임워크
  • 메타데이터 추가없이 존재하는 Grails 애플리케이션 열기

Ruby and Rails

  • 테스트결과를 보여주는 새로운 태스트 실행 인터페이스
  • Rake 향상: Rake Runner와 미리생성된 Rake 파일
  • 디버거: 조건별 브레이크 포인트와 캐치포인트
  • 명령줄에서 JVM 옵션을 허용하는 Ruby projects

웹개발을 위한 GlassFish v3 Prelude

  • 모듈방식, OSGi 기반의 아키텍쳐
  • 작은 footprint, 빠른 시작과 설치(deploy)
  • JRuby를 포함한 스크립팅 지원

C/C++

  • 향상된 코드완성, 오류 강조와 문법 강조
  • 콜 그래프, 메모리 창
  • 원격 개발
  • tar파일, zip파일 또한 SVR4 페키지로 어플리케이션 페키징


Java ME

  • 디자이너 팔레트에 사용자지정 컴포넌트를 추가할 수 있는 새로운 프로젝트 마법사
  • New SVG UI components in the enhanced SVG Composer enable rich UI design
  • New Data Binding Custom Components in the Visual Designer Palette
  • Upgraded obfuscation tool to ProGuard 4.2 and test framework to JMUnit 1.1.0

Java 디버거

  • 향상된 UI와 업무절차로 무장된 새로운 다중 뜨레드 디버깅
  • 똑똑한 한걸음(Step Into)과 더불어 식내에서 메소드 선발
  • 교착 탐지를 시각적으로 표시


Java SE

  • 저장시 자동 컴파일
  • 향상된 Eclipse 프로젝트 가져오기와 동기화
  • 자바 호출 계층
  • Javadoc 분석
  • 간단하거나 적격인 클래스 이름을 생성할 수 있는 Swing GUI 빌더


IDE 도구와 사용성

  • CamelCase 코드 완성
  • 프로젝트별 사용자지정 포멧팅
  • 파일/심불/타입, IDE 액션, 옵션과 문서를 위한 IDE전역 빠른검색 단축키
  • 윈도우 UNC경로 지원 (네트워크 공유 폴더)
