Информация к выпуску среды IDE NetBeans 7.0

IDE NetBeans представляет собой отмеченную наградами среду разработки, доступную для ОС Windows, Mac, Linux и Solaris. Проект NetBeans состоит из среды IDE с открытым исходным кодом и платформы приложения, которая позволяет разработчикам ускорить работу над созданием настольных и мобильных приложений, а также веб-приложений и приложений уровня предприятия, используя при этом платформу Java, PHP, JavaScript и Ajax, Groovy и Grails, а также C/C++.

Проект NetBeans поддерживается активным сообществом разработчиков, которое предлагает обширную документацию и обучающие материалы, а также множество сторонних подключаемых модулей.

Обзор выпуска

Среда IDE NetBeans 7.0 предоставляет поддержку языков для разработки в соответствии с предлагаемой спецификацией Java SE 7 при помощи предварительного просмотра JDK 7. Кроме того, данный выпуск предоставляет расширенные возможности интеграции с сервером Oracle WebLogic, а также поддержку Oracle Database и GlassFish 3.1. Дополнительные функции включают в себя поддержку редактирования с помощью Maven 3 и HTML5, новое средство разработки GridBagLayout с расширенными возможностями разработки GUI Swing, дополнительные функции редактора Java и др.

Среда IDE NetBeans 7.0 доступна на английском, португальском, японском, русском и упрощенном китайском.

Ключевые возможности

JDK 7

  • Поддержка проекта Coin
  • Новые функции редактора: автозавершение кода, подсказки

WebLogic Server

  • Оптимизированный и быстрый процесс разработки на WebLogic
  • Новый узел среды выполнения сервера с развернутыми приложениями и ресурсами
  • Интеграция JSF с серверными библиотеками

Oracle Database

  • Мастер быстрого подключения
  • Обзор установки для драйвера JDBC
  • Редактирование и разработка сохраненных процедур




GlassFish

  • Поддержка GlassFish 3.1
  • Перезагрузка домена и просмотр журнала для удаленного сервера GlassFish
  • Включение и отключение развернутых приложений

Java

  • Поддержка Maven 3
  • Интеграция JUnit 4.8.2 и усовершенствованные функции JUnit
  • Удаленные HTTP URL, поддерживаемые для документации Javadoc в библиотеках и платформах Java
  • Новое усовершенствованное средство визуальной настройки GridBagLayout

Java EE

  • Оптимизированная поддержка CDI, служб REST и Java Persistence
  • Новые возможности поддержки для проверки компонентов
  • Поддержка библиотек компонентов JSF, включая встроенную библиотеку PrimeFaces
  • Усовершенствованные возможности редактирования для языка выражений JSF, включая автозавершение кода, реорганизацию и подсказки





Веб-языки

  • Поддержка редактирования HTML5
  • Средство форматирования JSON

PHP

  • Создание PhpDoc
  • Реорганизация с переименованием, реорганизация с безопасным удалением
  • PHP 5.3 - поддержка псевдонимов

C/C++

  • Упрощенный импорт проекта из существующего пользовательского двоичного кода
  • Новый тип проекта, в котором исходные файлы пользователя расположены в удаленной системе

Платформа NetBeans

  • Аннотации для создания регистрации действия в слое
  • Улучшенная производительность и тесная интеграция с профайлером
  • Дополнительные изменения API-интерфейса NetBeans

Общие возможности

  • Перенос по словам в редакторе
  • Расширенные возможности интеграции с профайлером
  • Упрощенная проверка на внешние изменения при переключении между средой IDE и другими программами.
