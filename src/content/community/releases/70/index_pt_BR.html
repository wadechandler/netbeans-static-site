Informação da versão do NetBeans IDE 7.0

O NetBeans IDE é um premiado ambiente de desenvolvimento integrado disponível para Windows, Mac, Linux e Solaris. O projeto NetBeans consiste em um IDE de código aberto e uma plataforma de aplicativos que permite aos desenvolvedores criar rapidamente aplicativos Web, empresariais, móveis e de área de trabalho usando a plataforma Java, bem como PHP, JavaScript e Ajax, Groovy e Grails, e C/C++.

O projeto NetBeans tem como suporte uma entusiasmada comunidade de desenvolvedorese oferece vastos recursos como documentação e treinamento, bem como uma variada seleção de plug-ins de terceiros.

Visão geral da versão

O NetBeans IDE 7.0 introduz o suporte à linguagem para o desenvolvimento da especificação com o JDK 7 Developer Preview. Essa versão também oferece integração aprimorada com o servidor Oracle WebLogic, bem como suporte ao banco de dados Oracle e GlassFish 3.1. Outros destaques são: o suporte à edição de HTML5 e Maven 3, o novo designer GridBagLayout para melhor desenvolvimento com a GUI Swing, aprimoramentos no editor Java e mais.

O NetBeans IDE 7.0 está disponível em inglês, português do Brasil, japonês, russo e chinês simplificado.

Recursos em destaque

JDK 7

  • Suporte ao projeto Coin
  • Aprimoramentos no editor: auto-completar código, dicas

Servidor WebLogic

  • Depolyment mais rápido e simplificado no WebLogic
  • Novo nó de tempo de execução do servidor que exibe recursos e aplicativos implantados
  • Integração JSF com bibliotecas de servidor

Banco de dados Oracle

  • Assistente de conexão simplificada
  • Instalação guiada do driver JDBC
  • Edição e deployment de procedimentos armazenados




GlassFish

  • Suporte a GlassFish 3.1
  • Reinício de domínio e visualizador de log para GlassFish remoto
  • Ativar e desativar aplicativos implantados

Java

  • Suporte a Maven 3
  • Integração JUnit 4.8.2 e vários aprimoramentos no JUnit
  • URLs HTTP remotos suportados para Javadoc em bibliotecas e plataformas Java
  • Novo personalizador visual aprimorado para GridBagLayout

Java EE

  • Suporte aprimorado para CDI, serviços REST e Java Persistence
  • Novo suporte para Bean Validation
  • Suporte para bibliotecas de componente para JSF, incluindo a biblioteca PrimeFaces incorporada
  • Edição aprimorada para linguagem de expressão no JSF, incluindo autocompletar de código, refatoração e dicas





Linguagens Web

  • Suporte à edição de HTML5
  • Formatador JSON

PHP

  • Gerar PhpDoc
  • Refatoração Renomear, refatoração Excluir com segurança
  • PHP 5.3 - Suporte a aliases

C/C++

  • Fácil importação de projetos do binário existente do usuário
  • Novo tipo de projeto em que os códigos-fonte do usuário são colocados no sistema remoto

Plataforma NetBeans

  • Anotações para gerar registros das ações na camada
  • Aprimoramentos no desempenho e integração precisa com o Profiler
  • Alterações adicionais na API do NetBeans

Geral

  • Quebra de linha no Editor
  • Integração aprimorada com o Profiler
  • Verificação menos intrusiva de alterações externas ao alternar entre o IDE e outros programas.
