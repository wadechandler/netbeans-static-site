Instruções de instalação do NetBeans IDE 7.0

Última atualização: 15 de abril de 2011

Este documento descreve como instalar o IDE NetBeans 7.0 em seu sistema. Consulte as Notas de Versão do NetBeans IDE 7.0 para obter informações sobre os sistemas operacionais suportados e sobre as configurações de hardware para o IDE. Para saber mais sobre os novos recursos incluídos nessa versão do IDE, consulte a página Informações da versão 7.0 do NetBeans IDE.

Software necessário

É necessário ter o Java SE Development Kit (JDK) 6, atualização 24, ou posterior para instalar o NetBeans IDE. A versão 7.0 do IDE não pode ser instalada utilizando o JDK 5.0. É possível efetuar o download da atualização mais recente do JDK 6 em http://www.oracle.com/technetwork/java/javase/downloads.

Os conjuntos PHP e C/C++ NetBeans somente requerem que o Java Runtime Environment (JRE) 6 seja instalado e executado. No entanto, se tiver a intenção de utilizar qualquer recurso Java, o JDK 6 será necessário.

Opções de download do Instalador

Na página de download do NetBeans IDE, você pode adquirir um dos diversos instaladores, cada um contendo o IDE básico e as ferramentas adicionais.

Os seguintes instaladores estão disponíveis:

  • Java SE. Oferece suporte a todos os recursos de desenvolvimento padrão Java SE, bem como à plataforma de desenvolvimento NetBeans RCP.
  • Java EE. Fornece ferramentas para o desenvolvimento de aplicativos Java SE e Java EE, assim como o suporte para a plataforma de desenvolvimento NetBeans RCP. Esta opção de download inclui também o GlassFish Server Open Source Edition 3.1 e o software Apache Tomcat 7.0.11.
  • C/C++. Oferece suporte para o desenvolvimento nas linguagens C, C++, Fortran e Assembly.
  • PHP. Fornece ferramentas para o desenvolvimento PHP 5.x e o suporte para o Zend e Symfony Framework.
  • Tudo. Trata-se de uma opção de download completo que contém todos os executáveis e tecnologias disponíveis para o NetBeans IDE.

Personalizando sua instalação

Se você fez o download da opção "Tudo", pode escolher exatamente as ferramentas e os tempos de execução para a instalação. Clique em Personalizar na página de boas-vindas do instalador e selecione os recursos que deseja utilizar. Para obter mais informações, consulte Instalando o software.

Se você deseja adicionar funcionalidade a sua configuração do NetBeans IDE, utilize o NetBeans Plugin Manager. Por exemplo, suponhamos que você começou baixando e instalando o pacote PHP. Em seguida, você decidiu que deseja experimentar a funcionalidade C/C++. Para fazer isso, acesse o NetBeans Plugin Manager no IDE (selecione Ferramentas > Plug-ins) e adicione o pacote C/C++ à instalação existente.

Há diversas notas importantes sobre a instalação e a configuração de alguns recursos do NetBeans IDE:

  1. O Apache Tomcat 7.0.11 está incluído nas opções de download "Java EE" e "Tudo", mas não está instalado por padrão nessas opções. Para instalar o Apache Tomcat, selecione abaixo a opção apropriada:
    • no download do "Java EE", inicie o instalador e selecione a caixa de verificação Apache Tomcat 7.0.11 na página de boas-vindas;
    • na opção de download "Tudo", inicie o instalador e selecione Apache Tomcat 7.0.11 na caixa de diálogo Instalação personalizada.
  2. Para o desenvolvimento de C/C++ (aplicável aos downloads "C/C++" e "Tudo"), é necessário, além do mais, instalar os compiladores e as ferramentas. Para obter mais informações, consulte Configurando o NetBeans IDE para C/C++/Fortran.
  3. Para Mac OS X, é possível personalizar a instalação do NetBeans IDE na opção download "Tudo". A instalação do IDE nas opções de download "Java SE", "Java EE", "C/C++" e "PHP" não pode ser personalizada.
  4. Também é possível fazer o download de um arquivo zip independente da plataforma e executar o arquivo executável do NetBeans IDE. Observe que o GlassFish Server Open Source e o Apache Tomcat estão disponíveis somente com instaladores específicos da plataforma.

Iniciando o download

  1. Vá para https://netbeans.org/downloads/
  2. Na parte superior direita da página, selecione a linguagem e a plataforma na lista suspensa. Também é possível optar por fazer o download e utilizar somente o arquivo zip independente da plataforma.
  3. Clique no botão Download para selecionar a opção de download que deseja instalar.
  4. Salve o arquivo do instalador no sistema.

Instalando o software

Siga as instruções nesta seção para instalar o IDE no sistema. Estas instruções de instalação aplicam-se a todas as plataformas compatíveis. Para obter uma lista das plataformas compatíveis e dos requisitos do sistema, consulte as notas de versão.

Microsoft Windows, Solaris OS e Linux

Para instalar o software:

  1. após concluir o download, execute o instalador.
    • Para Windows, o arquivo executável do instalador tem a extensão .exe. Clique duas vezes no arquivo do instalador para executá-lo.
    • Nas plataformas Solaris e Linux, o arquivo do instalador tem a extensão .sh. Para essas plataformas, é preciso executar os arquivos do instalador com o seguinte comando: chmod +x <installer-file-name>
  2. Se tiver optado pela opção de download "Tudo", você pode personalizar a sua instalação. Execute as seguintes etapas na página de boas-vindas do assistente de instalação:
    1. clique em Personalizar.
    2. Na caixa de diálogo Personalizar a instalação, faça suas seleções.
    3. Clique em OK.
  3. Clique em Avançar na página de boas-vindas do assistente de instalação.
  4. Na página Contrato de Licença, leia o contrato de licença, clique na caixa de seleção para aceitar e clique em Avançar.
  5. Na página Contrato de licença do JUnit, decida se deseja instalar o JUnit, clique na opção apropriada e clique em Avançar.
  6. Na página de instalação do IDE NetBeans, proceda da seguinte maneira:
    1. aceite o diretório padrão de instalação do IDE NetBeans ou especifique outro diretório.

      Nota: o diretório de instalação deverá estar vazio e o perfil de usuário que estiver sendo utilizado para executar o instalador deverá ter permissões de leitura/gravação para esse diretório.

    2. Aceite a instalação padrão do JDK para utilizar com o NetBeans IDE ou selecione uma instalação diferente na lista suspensa. Se o assistente de instalação não encontrou uma instalação JDK compatível para ser utilizada com o NetBeans IDE, o JDK não está instalado no local padrão. Nesse caso, especifique o caminho do JDK instalado e clique em Avançar ou cancele a instalação atual. Após a instalação da versão necessária do JDK, você pode reiniciar a instalação.
  7. Se a página de instalação do GlassFish Server Open Source Edition 3.1 for aberta, aceite o diretório padrão de instalação ou especifique outro local de instalação.
  8. Se estiver instalando o Apache Tomcat, em sua página de instalação, aceite o diretório padrão de instalação ou especifique outro local de instalação. Clique em Próximo.
  9. Na página Resumo, verifique se a lista de componentes a serem instalados está correta e se você tem espaço suficiente no sistema para a instalação.
  10. Clique em Instalar para iniciar a instalação.
  11. Na página Configuração concluída, forneça os dados de utilização anônimos, se desejado, e clique em Finalizar.

Nota: se você encontrar problemas após concluir instalação do software com êxito, consulte Solução de problemas para obter descrições e soluções para as questões não-resolvidas que poderão afetar o processo de instalação.

Mac OS X

Para instalar o software:

  1. Após concluir o download, execute o instalador. O arquivo do instalador tem a extensão .dmg.
  2. Clique no ícone de pacote no painel exibido. A extensão do pacote é .mpkg . O assistente de instalação é iniciado.
  3. Na página de boas-vindas do assistente de instalação, clique em Continuar.
  4. Leia o contrato de licença e clique em Continuar. Clique em Aceitar na janela suspensa para aceitar a licença.
  5. Na página Selecionar um destino, selecione o drive e clique em Continuar.
  6. Se tiver optado pela opção de download "Tudo", você pode personalizar a sua instalação. Pressione o botão Personalizar no lado esquerdo inferior da janela do assistente de instalação, na última janela do assistente de instalação. A árvore de produtos é exibida.
  7. Selecione os produtos que você deseja instalar.
  8. Para iniciar a instalação, digite o nome do administrador e a senha no sistema e clique em OK.

Pacote independente de plataforma

Para executar o software:

  1. após o download do arquivo ZIP independente da plataforma, extraia-o em qualquer pasta do sistema.
  2. Execute o arquivo executável localizado no diretório netbeans/bin.
  3. Aceite o Contrato de Licença.
  4. Na página Contrato de licença do JUnit, decida se deseja instalar o JUnit, clique na opção apropriada e clique em Avançar. A instalação do NetBeans IDE se inicia.

Nota: se não for possível encontrar a instalação do JDK compatível, talvez seja necessário adicionar manualmente o caminho ao diretório de instalação do JDK, adotando o seguinte procedimento:

  1. abra o arquivo netbeans.conf localizado no diretório netbeans/etc em um editor de texto.
  2. Insira a localização da instalação do JDK compatível para a opção netbeans_jdkhome. A localização padrão no Windows é C:\Program Files\Java\jdk1.6.0_24.
  3. Salve o arquivo netbeans.conf e execute o arquivo executável no diretório netbeans/bin.

Nota: o GlassFish Server Open Source Edition e o Apache Tomcat somente estão disponíveis com instaladores específicos da plataforma.

Nota: para utilizar o GlassFish Server Open Source Edition e/ou Apache Tomcat que já estão instalados na máquina, no NetBeans IDE, clique em Servidores no menu Ferramentas. Isso exibe o Assistente para adicionar instância de servidor que ajuda a registrar uma instância do servidor Web necessário no NetBeans IDE.

Desinstalando o software

Siga as instruções nesta seção para desinstalar o IDE de seu sistema. Estas instruções de desinstalação aplicam-se a todas as plataformas compatíveis. Para obter uma lista das plataformas compatíveis, consulte as notas de versão.

Microsoft Windows

Se necessário, você pode desinstalar o NetBeans IDE, o GlassFish Server Open Source 3.1 e o Apache Tomcat utilizando os seguintes procedimentos.

Para desinstalar o IDE:

  1. Feche o IDE.
  2. No Painel de controle, selecione Adicionar ou remover programas e então selecione IDE NetBeans 7.0 e clique em Remover.
  3. Para desinstalar o GlassFish, o Tomcat e o IDE ao mesmo tempo, na página Resumo assegure-se de que as versões listadas são as corretas e as caixas apropriadas estão marcadas.
  4. Marque a caixa de verificação no diretório do usuário do NetBeans se desejar removê-lo junto com o IDE e clique em Desinstalar.
  5. Na página Resumo, clique em Desinstalar.
  6. Quando a desinstalação for concluída, clique em Concluir.

Para desinstalar o GlassFish Server Open Source Edition 3.1:

  1. Feche o IDE.
  2. No Painel de controle, selecione Adicionar ou remover programas e, em seguida, selecione GlassFish Server Open Source Edition 3.1 e clique em Remover.

Para desinstalar o Apache Tomcat:

  1. Feche o IDE.
  2. No Painel de controle, selecione Adicionar ou remover programas e, em seguida, selecione Apache Tomcat 7.0.11 e clique em Remover.

Solaris OS, Linux

Se necessário, você pode desinstalar o NetBeans IDE, o GlassFish Server Open Source 3.1 e o Apache Tomcat utilizando os seguintes procedimentos.

Para desinstalar o IDE:

  1. feche o IDE.
  2. No diretório de instalação do IDE, execute o arquivo uninstall.sh.
  3. Para desinstalar o GlassFish, o Tomcat e o IDE ao mesmo tempo, na página Resumo assegure-se de que as versões listadas são as corretas e as caixas apropriadas estão marcadas.
  4. Na página Resumo, clique em Desinstalar.
  5. Quando a desinstalação for concluída, clique em Concluir.

Para desinstalar o GlassFish Server Open Source Edition 3.1 :

  1. Feche o IDE.
  2. Localize o diretório de instalação do servidor de aplicações.

    Nota: o local padrão de instalação será diferente, dependendo do tipo de instalação e da plataforma. A tabela a seguir exibe o diretório padrão de instalação do servidor de aplicativos para várias plataformas.

    Plataforma Instalando como... Instalação do GlassFish 3.1
    Windows usuário com direitos de administrador %ProgramFiles%\glassfish-3.1
    Windows usuário %USERPROFILE%\glassfish-3.1
    Solaris OS  raiz /opt/glassfish-3.1
    Solaris OS, Linux usuário ~/glassfish-3.1
    Linux raiz /usr/local/glassfish-3.1
  3. No diretório de instalação do servidor de aplicações, execute o arquivo uninstall.sh.
  4. Na página Resumo, clique em Desinstalar.
  5. Quando a desinstalação for concluída, clique em Concluir.

Para desinstalar o Apache Tomcat:

  1. Feche o IDE.
  2. Localize o diretório de instalação do Apache Tomcat.

    Nota: o local padrão de instalação varia dependendo do tipo de instalação e da plataforma. A tabela a seguir exibe o diretório padrão de instalação do servidor de aplicativos em diferentes plataformas.

    Plataforma Instalando como... Instalação do Apache Tomcat 7.0.11
    Windows usuário com direitos de administrador %ProgramFiles%\Apache Software Foundation\Apache Tomcat 7.0.11
    Windows usuário %USERPROFILE%\Apache Software Foundation\Apache Tomcat 7.0.11
    Solaris OS  raiz /opt/apache-tomcat-7.0.11
    Solaris OS, Linux usuário ~/apache-tomcat-7.0.11
    Linux raiz /usr/local/apache-tomcat-7.0.11
  3. No diretório de instalação do servidor de aplicações, execute o arquivo uninstall.sh.
  4. Na página Resumo, clique em Desinstalar.
  5. Quando a desinstalação for concluída, clique em Concluir

Mac OS X

Para desinstalar o IDE e o servidor de aplicativos:

  1. feche o software.
  2. Arraste as pastas ou arquivos de programas para a Lixeira. Observe que o local padrão do IDE e do servidor de aplicações é /Applications/NetBeans/.

Pacote independente de plataforma

Para desinstalar os arquivos do IDE, siga as etapas descritas na seção Limpeza após instalações incompletas deste documento.

Atualização e suporte a várias instalações

Várias instalações do NetBeans IDE 5.x e 6.x podem coexistir com o NetBeans IDE 7.0 no mesmo sistema. Você não precisa desinstalar as versões anteriores para instalar ou executar o NetBeans IDE 7.0.

Se já tiver um NetBeans IDE instalado, terá a opção de importar ou não as configurações de usuário do diretório existente ao executar o NetBeans 7.0 pela primeira vez.

Nota: atualizações de versão incremental do IDE não podem ser executadas neste momento. Para atualizar para a versão mais recente, é preciso efetuar o download do instalador mais recente.

Solução de problemas

Sobre o diretório de trabalho do instalador

O instalador armazena dados sobre as sessões de instalação no diretório de trabalho do instalador. Exemplos desses dados são a lista de componentes instalados no sistema, as mensagens do registro de instalação e os dados de registro.

O local padrão do diretório de trabalho do instalador é %USERPROFILE%\.nbi no Windows e ~/.nbi em outras plataformas. O subdiretório do registro contém os arquivos de registro de instalação.

Limpeza após instalações incompletas

Após instalações incompletas ou malsucedidas, talvez seja necessário limpar o sistema das instalações anteriores.

Para Microsoft Windows, Solaris OS e Linux:

  1. exclua o diretório de trabalho do instalador.
  2. Exclua os diretórios de instalação do IDE e do servidor de aplicações GlassFish.
  3. Na plataforma Windows, exclua todas as entradas do registro menos a nbi* de todas as chaves de registro: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall

Para Mac OS X:

  • arraste as pastas ou arquivos de programas para a Lixeira.

Desabilitando alguns produtos de software antes da instalação

É melhor desabilitar o firewall, o antivírus e os aplicativos de monitoramento antes de iniciar a instalação. Alguns desses aplicativos podem evitar que os componentes do IDE sejam configurados corretamente durante o processo de instalação.

Por exemplo, se você possuir o aplicativo Logitech Process Monitor (LVPrcSrv.exe) no sistema, desative-o antes da instalação.
