NetBeans Mobility Pack for CDC 5.5.1 安装指南

NetBeans Mobility Pack for CDC 5.5.1 提供了一个用于快速构建 Java Micro Edition (Java ME) 应用程序的环境，所构建的应用程序支持那些使用联网的设备配置 (Connected Device Configuration, CDC) 以及 Personal、Foundation 和 AGUI Profile 的设备。

注意：有关本软件包中所包含软件的系统要求信息，请参见 NetBeans Mobility Pack for CDC 5.5.1 发行说明。

先决条件

本主题指定了在开始安装 NetBeans Mobility Pack for CDC 5.5.1 之前必须满足的先决条件。

请注意，无法在 NetBeans IDE 5.5 上运行 NetBeans Mobility Pack for CDC。您必须使用 NetBeans IDE 5.5.1。

请通过 netbeans.org 下载页面下载 NetBeans Mobility Pack for CDC 5.5.1。

所需的软件

NetBeans Mobility Pack for CDC 5.5.1 发行说明详细介绍了硬件、软件和操作系统要求。

您需要先安装以下软件，然后再安装 NetBeans Mobility Pack for CDC 5.5.1。

  • Java SE Development Kit (JDK) 5.0 或 6.0
  • NetBeans IDE 5.5.1

NetBeans Mobility Pack 中集成了 Sun Java Toolkit for CDC。但是，如果要为其他支持的平台编写应用程序，则需要下载并安装以下一个或多个 SDK：

  • Ricoh
    • Embedded Software Architecture Emulator（下载时要求注册）
  • Sony Ericsson
    • UIQ 3 SDK（下载时要求登录）
    • M600 Extension Package for the UIQ 3 SDK Beta 3（下载）
    • P990 Extension Package for the UIQ 3 SDK Beta 3（下载）
  • Nokia
    • Nokia Series 80 Platform SDK for Symbian OS, for Java, Personal Profile（下载）
  • NSIcom (CrEme for Windows CE)
    • CrEme 4.10 Developer Support（下载）
    • Microsoft Device Emulator 1.0（下载）
    • Virtual Machine Network Driver for Microsoft Device Emulator（下载)
    • ActiveSync（下载）
  • SavaJe
    • SavaJe Mobile Platform（不再提供）

安装软件

下面是有关安装 NetBeans Mobility Pack for CDC 5.5.1 的说明。这些指导说明假定您已安装了 Java SE Development Kit 和 NetBeans IDE 5.5.1。

安装 NetBeans Mobility Pack for CDC 5.5.1：

  1. 访问 http://services.netbeans.org/downloads/index.php?p=10 站点以下载安装文件。
  2. 在系统中找到安装文件的位置，然后双击此安装程序的图标以启动安装程序。
  3. 在安装向导的“欢迎”页中，单击“下一步”。
  4. 接受许可证协议，然后单击“下一步”。
  5. 在安装向导的安装目录页中，指定 NetBeans IDE 5.5.1 的安装位置。
  6. 安装向导将搜索兼容的 JDK 安装。如果 JDK 没有位于缺省位置，则需要指定 Java JDK 1.5.0 或更高版本的位置。您也可以键入其他 JDK 的安装路径，或者使用“浏览”按钮导航并选择其他安装。
  7. 安装完成后，单击“完成”退出向导。

缺省的项目目录为 C:\Documents and Settings\username。当卸载 NetBeans Mobility Pack for CDC 时，并不会删除此目录，而且在下次安装 Mobility Pack for CDC 时，系统可以识别此目录。

注意：如果在软件安装过程中遇到问题，请参见疑难解答，以获取影响安装过程的待解决问题的完整描述以及建议的解决方法。

 

在非 Windows 平台上安装 NetBeans Mobility Pack for CDC 5.5.1

目前，Mobility Pack for CDC 由于缺少其他平台的第三方仿真器支持，不能用于非 Windows 平台。但是，对于那些希望将此软件包安装在其他操作系统上的用户，我们仍然提供了一个二进制 zip 文件。请注意，关于此二进制 zip 文件，我们未提供任何文档或支持。有关下载并安装二进制 zip 文件的信息，请访问维基页面以获取有关此主题的内容。您还可以通过 Mobility 用户邮件列表了解与此主题有关的更多问题。

安装仿真器平台

要开发 CDC 应用程序，您需要在 IDE 中注册 CDC 平台。下面的说明介绍了 IDE 当前支持的各种仿真器平台的安装过程。

Sun Java Toolkit for CDC

Sun Java Toolkit for CDC 与 NetBeans Mobility Pack 集成在一起，因此不需要下载并安装此仿真器。

但是，如果确实需要单独安装 Toolkit，则可以按照以下说明执行操作：

  1. 通过 Sun Developer Network 下载 Sun Java Toolkit for CDC 1.0（下载）。
  2. 按照提供的说明安装平台仿真器。
  3. 从主菜单中选择“工具”>“Java 平台管理器”。
  4. 单击“添加平台”。将打开“添加 Java 平台”对话框。
  5. 选择“Java Micro Edition CDC 平台仿真器”，然后单击“下一步”。
  6. 选择包含平台 SDK 的文件夹。缺省目录为 C:\CDCTK10\。单击“下一步”。
  7. 如果需要，请使用“平台名称”对话框来添加或删除其他资源库。
  8. 单击“完成”。IDE 将注册此 CDC 平台。

Ricoh Embedded Software Architecture

Mobility Pack for CDC 支持 Ricoh Embedded Software Architecture。要集成对此体系结构的支持，请按照以下说明执行操作：

  1. 通过 Ricoh Developer Program Site Web 站点下载以下文件：
    • Embedded Software Architecture Emulator（下载时要求登录）
  2. 解压缩 Embedded_Software_Architecture_Emulator_1.14c.zip 文件。
    这些文件将被解压缩至 <root>\Embedded_Software_Architecture_Emulator_1.14c\ 文件夹中。
  3. 从 NetBeans IDE 主菜单中选择“工具”>“Java 平台管理器”。
  4. 单击“添加平台”。将打开“添加 Java 平台”对话框。
  5. 选择“Java Micro Edition CDC 平台仿真器”，然后单击“下一步”。
  6. 选择包含平台 SDK 的文件夹。缺省目录为 C:\Embedded_Software_Architecture_Emulator_1.14c\。单击“下一步”。
  7. 如果需要，请使用“平台名称”对话框来添加或删除其他资源库。
  8. 单击“完成”。IDE 将注册此 CDC 平台。

Sony Ericsson CDC Platform for the UIQ SDK 3

Mobility Pack for CDC 支持 Sony Ericsson CDC Platform 1。要集成对 M600 和 P990 系列设备的支持，请按照以下说明执行操作：

  1. 通过 developer.sonyericsson.com Web 站点下载以下文件：
    • UIQ 3 SDK（下载时要求登录）
    • M600 Extension Package for the UIQ 3 SDK（下载）
    • P990 Extension Package for the UIQ 3 SDK（下载）
  2. 解压缩 UIQ3SDK_final.zip 文件。
  3. 在系统上找到这些文件的解压缩位置，然后双击 UIQS3SDK.exe。
    将打开 UIQ 3 SDK 安装向导。
  4. 按照向导的说明安装此 SDK。
    缺省安装目录为 C:\Symbian\UIQ3SDK\。
  5. 要安装 M600 和 P990 扩展包，请在系统上找到下载位置，然后双击安装程序图标以启动安装向导。
  6. 按照向导的说明安装每个 SDK。
  7. 从 NetBeans IDE 主菜单中选择“工具”>“Java 平台管理器”。
  8. 单击“添加平台”。将打开“添加 Java 平台”对话框。
  9. 选择“Java Micro Edition CDC 平台仿真器”，然后单击“下一步”。
  10. 选择包含平台 SDK 的文件夹。缺省目录为 C:\Symbian\UIQ3SDK\。单击“下一步”。
  11. 如果需要，请使用“平台名称”对话框来添加或删除其他资源库。

    注意：如果出现“未正确配置平台。请使用 SDK 配置工具来安装必需的工具。”消息时，表明未选择仿真器样式。

    要修复此问题，请执行以下操作：
    1. 单击“配置”标签，然后单击“SDK 配置工具”按钮。
    2. 在“SDK 配置”中，单击“样式”标签，然后从“仿真器样式”中选择一种样式。选择“应用样式”。
    3. 关闭 SDK 配置器。
  12. 单击“完成”。IDE 将注册此 CDC 平台。

Nokia Series 80 Platform SDK for Symbian OS, for Java, Personal Profile

Mobility Pack for CDC 支持 Series 80 Platform SDK for Symbian OS, for Java, Personal Profile。要集成对 Nokia Series 80 设备的支持，请按照以下说明执行操作：

  3. 在系统上找到该文件的解压缩位置。双击 setup.exe 文件图标。
    将打开 Nokia 安装程序向导。
  4. 按照向导的说明安装此 SDK。
    该向导将提示您把选择的设备作为缺省设备。选择 "Series80_DP2_0_PP_SDK"。
  5. 从 NetBeans IDE 主菜单中选择“工具”>“Java 平台管理器”。
  6. 单击“添加平台”。将打开“添加 Java 平台”对话框。
  7. 选择“Java Micro Edition CDC 平台仿真器”，然后单击“下一步”。
  8. 选择包含平台 SDK 的文件夹。缺省目录为 C:\Symbian\S80_DP2_0_PP_SDK。单击“下一步”。
  9. 如果需要，请使用“平台名称”对话框来添加或删除其他资源库。

CrEme VM for Windows CE

Mobility Pack for CDC 支持 NSIcom CrEme VM for Windows CE。要集成对 Windows CE 设备的支持，请按照以下说明执行操作。

  1. 通过 NSIcom 下载 CrEme 4.10 平台（下载）。

  4. 从 NetBeans IDE 主菜单中选择“工具”>“Java 平台管理器”。
  5. 单击“添加平台”。将打开“添加 Java 平台”对话框。
  6. 选择“Java Micro Edition CDC 平台仿真器”，然后单击“下一步”。
  7. 选择包含平台 SDK 的文件夹。缺省目录为 C:\Program Files\NSIcom\CrE-ME V4.10。单击“下一步”。
  8. 如果需要，请使用“平台名称”对话框来添加或删除其他资源库。
  9. 单击“完成”。IDE 将注册此 CDC 平台。

Microsoft Device Emulator 1.0

Mobility Pack for CDC 支持 Microsoft Device Emulator 1.0 以及 NSIcom CrEme VM for Windows CE。要集成 Microsoft Device Emulator 1.0，请按照以下说明执行操作。

  1. 下载此仿真器的 V1Emulator.zip 文件（下载）。
  2. 解压缩此文件，然后双击 standalone_emulator_V1.exe。设备仿真器的安装向导将指导您完成安装过程。
  3. 下载 efp.msi（下载）。此设备仿真器的安装向导将指导您完成安装过程。
  4. 您可能还需要安装 Virtual Machine Network Driver for Microsoft Device Emulator（下载）。下载该文件，然后双击 netsvwrap.msi 文件。该安装向导将指导您完成安装过程。

SavaJe Mobile Platform

Mobility Pack for CDC 支持通过 SavaJe Operating System 来创建 Xlet。Sun Java Wireless Toolkit for CDC 中内置了对 SavaJe OS 的支持。因为 Jasper S20 当前仅提供了预览版本，所以 Jasper S20 没有可用的仿真器样机。您可以改用 DefaultColorPhone 设备来模拟 Jasper S20 的行为。

如果您有一个 Jasper S20 设备，下面这些说明介绍了如何将应用程序部署到该设备上。您需要具备：

  • Jasper S20 电话
  • SavaJe Operating System (OS)（不再提供）
  • GSPDA 移动设备软件（MCCI USB 驱动程序）

设置手机

以下步骤说明了在 Jasper S20 电话上安装本教程所需软件的过程。

安装 SavaJE OS

注意：抱歉，SavaJe OS 不再提供可供下载的链接。这些指导说明只适用于那些通过 CD 或其他资源访问 SavaJe Mobile Platform 软件的用户。该软件是针对 Jasper S20 手机（安装了 SavaJe OS 早期版本）编写的，允许您保留现有的首选项设置和出厂设置。

访问 www.SavaJe.com 可以获得三个 SavaJe OS 版本：

  • JasperS20.zip
    电话附带提供的商业版。
  • JasperS20_EA_Developer.zip
    未启用 Java 调试有线协议 (Java Debug Wire Protocol, JDWP) 的 JavaOne Early Access 开发版。
  • JasperS20_EA_Developer_debug.zip
    启用 JDWP 的 JavaOne Early Access 开发版。要运行具有 JDWP 调试功能的软件包，您需要装入此版本。

 

执行以下步骤，在电话上安装更新的 SavaJE OS 版本。

  1. 将包含所需版本的 zip 文件解压缩至 mini-SD 卡的根目录中。
  2. 将 USB 电缆或电源电缆连接到手机上。
  3. 在按下 Camera 键的同时接通手机电源。
  4. 在更新程序菜单屏幕上，向下移动导航键，直至突出显示 "Factory Reset" 并选择该条目。向右移动导航键以启动出厂重置。使用导航键导航到 "Yes"，然后向右移动导航键以选择 "Yes"。
    电话将会开始装入新的版本。安装完成后，可能会提示您按任意键以重新启动电话。
  5. 手机随后将重新启动并显示 Xplore 屏幕。如果在重新启动过程中显示电池充电屏幕，请按电源按钮以继续重新启动电话。

出厂重置后首次启动手机时，将会显示 "Language/Locale" 选择菜单。请使用导航键导航并按导航键按钮以进行选择。系统还会提示您设置时间。

注册 CDC 平台

请按照为 Sun Java Wireless Toolkit for CDC 1.0 提供的说明执行操作。

卸载软件

本部分包含从系统中卸载 NetBeans Mobility Pack for CDC 5.5.1 的信息。

  1. 关闭 IDE。
  2. 从“控制面板”中单击“添加或删除程序”，选择 "NetBeans Mobility Pack for CDC 5.5.1"，然后单击“更改/删除”。
  3. 按照卸载程序中的说明执行操作。

注意：您还可以通过 Windows 控制面板卸载任意或全部仿真器平台。

疑难解答

如果您在安装或运行软件时遇到问题，请参见下面的问题列表以获取相应的解决方法。有关已知问题的完整列表、系统要求以及配置信息，请参见发行说明。

如果您遇到发行说明中未列出的问题，或者希望提出改进某项功能的请求，请在 NetBeans 错误跟踪系统 IssueZilla (https://netbeans.org/community/issues.html) 中记录这些问题。对于 NetBeans Mobility Pack 问题，请选择 "mobility" 组件，选择版本 "5.5.1"，然后选择 "cdc pack" 子组件。如果您不知道相应的子组件名称，请选择“其他”。

如果您已登录到 netbeans.org，则可以直接进入 Mobility Pack 问题输入屏幕：https://netbeans.org/bugzilla/enter_bug.cgi?component=mobility&version=5.5.1。
