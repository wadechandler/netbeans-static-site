Pré-requisitos

Antes da instalação do NetBeans C/C++ Development Pack 5.5.1 atualização 1, o software abaixo deve ser instalado:

  • NetBeans IDE 5.5.1 ou NetBeans IDE 5.5

  • O JDK 5.0 ou JDK 6.0 deve estar instalado no sistema. Se não tiver uma dessas versões do JDK instalada, não é possível prosseguir com a instalação. Para obter detalhes, consulte o tópico Software necessário em Notas da versão.

Instalação do software

Essa seção apresenta as instruções para a instalação do NetBeans C/C++ Development Pack em todas as plataformas suportadas.

Windows

Encontram-se abaixo as instruções para a instalação do NetBeans C/C++ Development Pack na plataforma Microsoft Windows.

Antes de começar

A instalação na plataforma Microsoft Windows apresenta as seguintes restrições:

  • O instalador extrai os arquivos necessários para o diretório temporário padrão definido pela variável de ambiente TEMP. Se o espaço em disco no diretório temporário é menor que o necessário, a instalação pode falhar ou ser feita de forma incorreta.
    Se tiver este problema, você pode aumentar o espaço alocado da variável de ambiente TEMP ou especificar outro diretório temporário com espaço suficiente. Para especificar um diretório temporário para a instalação, use os seguintes comandos de uma janela de comando:

    > cd /d  local-do-instalador
    > netbeans-c++-5_5_1_u1-windows.exe -is:tempdir diretório-temporário

Para instalar

  1. Depois de fazer o download do arquivo de instalação netbeans-c++-5_5_1_u1-windows.exe, clique duas vezes no ícone do instalador para iniciá-lo.

  2. Na página de boas-vindas do assistente para instalação, clique em Próximo.

  3. Aceite o contrato de licença e clique em Próximo.

  4. Na página do diretório de instalação do assistente para instalação, aceite o diretório do NetBeans IDE encontrado pelo assistente ou selecione uma instalação diferente do IDE. Clique em Próximo.

  5. Verifique se o local de instalação está correto e se há espaço disponível no sistema para a instalação.

  6. Clique em Próximo para iniciar a instalação.

  7. Quando a instalação estiver concluída, clique em Finalizar para sair do assistente.

Solaris™ OS

Encontram-se abaixo as instruções para a instalação do NetBeans C/C++ Development Pack na plataforma Solaris OS.

Nas instruções abaixo, substitua seu_executável_binário por um dos seguintes:

  • Solaris™ OS, SPARC®: netbeans-c++-5_5_1_u1-solaris-sparc.bin
  • Solaris™ OS, x86/x64 Platform Edition: netbeans-c++-5_5_1_u1-solaris-x86.bin

    $ chmod +x seu_executável_binário
  3. Inicie o instalador usando o comando mostrado abaixo em um prompt de comando. 
    $ ./seu_executável_binário
  4. Na página de boas-vindas do assistente para instalação, clique em Próximo.

  5. Aceite o contrato de licença e clique em Próximo.

  6. Na página do diretório de instalação do assistente para instalação, aceite o diretório do NetBeans IDE encontrado pelo assistente ou selecione uma instalação diferente do IDE. Clique em Próximo.

  7. Verifique se o local de instalação está correto e se há espaço disponível no sistema para a instalação.

  8. Clique em Próximo para iniciar a instalação.

  9. Quando a instalação estiver concluída, clique em Finalizar para sair do assistente.

Instalando e configurando compiladores e ferramentas

O Netbeans C/C++ Development Pack requer um compilador C, um compilador C++, um utilitário make e um depurador gdb.

Windows

O NetBeans C/C++ Development Pack foi testado nos seguintes compiladores e ferramentas:

  • Cygwin 1.5.21
  • Cygwin gcc-core (compilador C) 3.4.x
  • Cygwin gcc-core (compilador C++) 3.4.x
  • Cygwin gdb (depurador GNU) 6.5.50
  • Cygwin make 3.80

Se já tiver instalados os compiladores Cygwin GNU, GNU make e o depurador gdb no sistema Windows e o caminho estiver configurado corretamente para encontrá-los, certifique-se de que as versões são corretas. Se as versões forem as corretas, não serão necessárias configurações adicionais.

Para verificar as versões das ferramentas e dos compiladores Cygwin:

  1. Verifique a versão do Cygwin digitando: 

    cygcheck -c cygwin
  2. Verifique as versões dos compiladores, do make e do gdb digitando: 
    gcc --version
g++ --version
make --version
gdb --version

Para instalar os compiladores GNU, o make e o depurador gdb de cygwin.com:

  1. Faça o download do programa setup.exe do Cygwin clicando no ícone Install or Update Now! (Instalar ou atualizar agora!) no meio da página.

  2. Execute o programa setup.exe. Aceite os padrões até encontrar a página Select Your Internet Connection (Selecionar conexão de Internet). Nessa página, selecione a opção que melhor se aplique ao seu caso. Clique em Próximo.

  3. Na página Choose Download Site (Selecionar local de download), escolha um local de download que esteja relativamente próximo a você. Clique em Próximo.

  4. Na página Select Packages (Selecionar pacotes), selecione os pacotes para download. Clique no sinal + ao lado de Devel para expandir a categoria das ferramentas de desenvolvimento. Se quiser exibir mais de uma janela ao mesmo tempo, redimencione-as.

  5. Selecione os pacotes que deseja clicando no rótulo Skip (Omitir) ao lado de tal pacote. Selecione como mínimo o gcc-core: o compilador C, gcc-g++: o compilador C++, gdb: O depurador GNU e o make: a versão GNU do utilitário 'make'.

  6. Agora adicione o diretório Compiler ao caminho:

    1. Abra o Painel de controle (Início > Configurações > Painel de controle) e clique duas vezes no programa Sistema.

    2. Selecione a guia Avançado e clique em Variáveis de ambiente.

    3. No painel Variáveis do sistema da caixa de diálogo Variáveis de ambiente, selecione a variável Path e clique em Editar.

    4. Adicione o caminho para o diretório cygwin-directory\bin à variável Path e clique em OK. Por padrão, cygwin-directory é C:\cygwin. Os nomes do diretório devem estar separados por ponto-e-vírgula.

    5. Clique em OK na caixa de diálogo Variáveis de ambiente e na caixa de diálogo Propriedades do sistema.

Solaris OS

O NetBeans C/C++ Development Pack foi testado nos seguintes compiladores e ferramentas:

  • Compilador C do Sun Studio 12 5.9 e gcc 3.4.3
  • Compilador C++ do Sun Studio 12 5.9 e g++ 3.4.3
  • gdb (depurador GNU) 6.2.1
  • Solaris make e gmake 3.80

Compiladores do Sun Studio 12

Se quiser usar os compiladores do Sun Studio 12:

  • Se tiver o software Sun Studio 12 instalado, certifique-se de que /installation directory/bin esteja no caminho antes de iniciar o NetBeans IDE.
  • Se não tiver o software Sun Studio 12 instalado, você pode fazer o download gratuitamente em http://developers.sun.com/sunstudio/downloads/.

Para fazer o download e instalar os compiladores do Sun Studio 12:

  1. Crie um diretório para o arquivo descarregado. Você deve ter permissões de escritura para este diretório.

  2. Faça o download do arquivo para a sua plataforma no diretório de download.

  3. Vá ao diretório de download e descompacte o arquivo descarregado. 
    bzcat nomedoarquivo | tar xvf -
  4. Siga as instruções do Capítulo 2 do guia Sun Studio 12 Quick Installation (English, Japanese, Simplified Chinese) para instalar os compiladores C, C++ e os patches necessários do Solaris.

Adicione o caminho para o software Sun Studio ao PATH antes de iniciar o NetBeans IDE.

Compiladores GNU e GNU make

Se quiser usar os compiladores GNU e o GNU make:

  • Se tiver uma instalação padrão do sistema operacional Solaris 10, os compiladores e o gmake são instalados em /usr/sfw/bin. Certifique-se de que este local está no seu caminho antes de iniciar o NetBeans IDE.
  • Se os compiladores e o gmake não estão instalados no sistema, é possível fazer o download em http://www.sunfreeware.com.

Para fazer o download e instalar os compiladores GNU e o make

  1. Faça o download de gcc 3.4.6 e make 3.81.

  2. Se os arquivos zip de download não são descompactados automaticamente durante o download, descompacte-os com o gunzip.

  3. Instale os pacotes com o comando pkgadd.

Certifique-se de incluir os diretórios do compilador GNU e do GNU make no caminho antes de iniciar o NetBeans IDE.

Depurador gdb

Tanto se usar os compiladores do Sun Studio e o Solaris make quanto se usar os compiladores GNU e o GNU make, você deve ter o depurador gbd para depurar os aplicativos no NetBeans C/C++ Development Pack. Você pode fazer o download do gdb 6.2.1 em ftp://ftp.gnu.org/gnu/gdb/.

Para fazer o download e instalar o gdb:

  1. Em "Select a Download" (Selecionar download) na parte inferior da página, selecione o download do Solaris 10 Companion Software para a plataforma Solaris.

  2. Na página de download, aceite o Contrato de licença e selecione o pacote gbd - GNU source level debugger.

  3. Torne-se raiz (superusuário).

  4. Descompacte o arquivo com binzip e instale o gbd com pkgadd: 
    bunzip SFWgdb.bz2
pkgadd -d SFWgbd

Certifique-se de incluir o caminho para o gdb antes de iniciar o NetBeans IDE.

Verificando a instalação

Para verificar se a instalação está correta, inicie o NetBeans IDE, crie um projeto de amostra e execute-o no depurador gdb.

Windows

Para iniciar o IDE em computadores com o Microsoft Windows, escolha uma das seguintes opções:

  • Clique duas vezes no ícone do NetBeans IDE na área de trabalho.

  • No menu Iniciar, selecione Programas > NetBeans 5.5.1 > NetBeans IDE ou Programa > NetBeans 5.5 > NetBeans IDE.

Para criar um projeto de amostra e executá-lo no depurador:

  1. Abra o assistente para Novo projeto em Arquivo > Novo projeto.

  2. No painel Categorias na página Escolher projeto do assistente, expanda a categoria Amostras e a subcategoria Desenvolvimento em C/C++ e selecione a subcategoria C/C++.

  3. Na janela Projetos, selecione o projeto Welcome. Clique em Próximo.

  4. Na página Nome e local do projeto, clique em Finalizar.

  5. Na guia Projetos do IDE, clique com o botão direito do mouse no projeto Welcome_1 e escolha Construir projeto. Se os compiladores e o utilitário make estiverem corretamente instalados e o caminho que leva a eles estiver definido, a saída de construção é exibida na janela Saída e a construção do projeto é bem-sucedida.

  6. Clique duas vezes em welcome.cc para abri-lo no Editor de código.

  7. Clique com o botão direito do mouse na margem esquerda da janela do Editor de código e escolha Mostrar números de linha.

  8. Defina o ponto de interrupção clicando na margem esquerda da janela do Editor de código ao lado da linha 33.

  9. Clique com o botão direito do mouse no projeto e escolha Depurar projeto. Se o depurador gdb estiver instalado corretamente e o caminho estiver definido, o gdb se inicia, as guias do depurador são exibidas e o aplicativo Welcome é executado e para no ponto de interrupção.

  10. Escolha Executar > Continuar para executar o aplicativo até o final.

Desinstalando o software

Windows

Para desinstalar o software:

  1. Feche o IDE.
  2. No Painel de controle, inicie Adicionar ou remover programas, selecione NetBeans C/C++ Development Pack 5.5.1 e clique em Alterar/Remover.
  3. Siga as instruções do desinstalador.

Solaris OS, Linux

Para desinstalar o software:

  1. Feche o IDE.
  2. Vá até o diretório /NetBeans-5.5.1-directory/cnd1/_uninst.
  3. Digite ./uninstaller para executar o desinstalador InstallShield.

Mac OS X

Para desinstalar o software:

  1. Feche o IDE.
  2. Vá até o diretório /~/Desktop/NetBeans.app/Contents/Resources/NetBeans////cnd1/_uninst. (Se o NetBeans IDE estiver instalado em outro diretório que não seja a Área de trabalho, substitua /~/Área de trabalho do caminho por tal diretório.)
  3. Digite ./uninstaller para executar o desinstalador InstallShield.
