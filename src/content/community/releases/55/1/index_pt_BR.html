Destaques dos recursos da versão 5.5.1





O principal recurso do NetBeans 5.5.1 é o suporte ao Java EE 5 Application Server 9.1. As construções beta do servidor de aplicativos 9.1 (GlassFish v2) estão agora disponíveis.

O NetBeans 5.5.1 também acrescenta o Windows Vista como plataforma suportada.

O Mobility Pack inclui o J2ME Wireless Toolkit, versão 2.5.1.

O NetBeans 5.5.1 foi estruturado sobre as funcionalidades do NetBeans 5.5 e também oferece várias correções a erros. Para ver uma lista completa dos recursos do NetBeans 5.5, consulte a Página da versão 5.5.

Pacotes suplementares do NetBeans

    Utilize o Mobility Pack para escrever, testar e depurar aplicativos dos dispositivos móveis oferecidos pela tecnologia da plataforma Java Micro Edition (plataforma Java ME).


    Use o Visual Web Pack para construir rápida e visualmente aplicativos da Web com base nos padrões, incluindo o suporte aos componentes AJAX e JSF.



    O NetBeans Enterprise Pack oferece suporte à criação de aplicativos SOA (Service Oriented Architecture). Observação: Se quiser usar a funcionalidade SOA e BPEL no Enterprise Pack, utilize o NetBeans IDE 5.5 com o NetBeans Enterprise Pack 5.5. Não temos previsto lançar futuras atualizações do NetBeans Enterprise Pack 5.x. Todos os nossos esforços estão voltados para o desenvolvimento do NetBeans 6.0 com SOA.


    O NetBeans Profiler ajuda você a otimizar a memória de aplicativos e o uso da CPU.



    O C/C++ Pack é compatível com projetos de C/C++. O editor que detecta as linguagens recua, completa e faz realce sintático do código do C/C++.


Outros idiomas

O NetBeans 5.5.1 foi traduzido ao chinês simplificado, japonês e português do Brasil - se estiver usando um destes locais, será automaticamente oferecida a você a versão traduzida, caso contrário escolha Outros sistemas & idiomas para localizar tais locais.
