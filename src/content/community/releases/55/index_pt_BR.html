Informações sobre o NetBeans IDE 5.5

A documentação do NetBeans é oferecida em guias baseados na Web e no sistema de ajuda da IDE. Você pode usar os documentos seguintes como introdução:

  • Notas de versão
  • Instruções de instalação
  • Documentação

O NetBeans 5.5 é compatível com a plataforma Java Enterprise Edition 5 e com as especificações Java Persistence, EJB 3 e JAX-WS 2.0.

O NetBeans IDE 5.5 amplia o sucesso da versão 5.0 e adiciona suporte para o Java EE 5 e o servidor de aplicativos Sun Java System PE 9. Já tem o Web 2.0? Claro! O NetBeans 5.5 gera artefatos JAX-WS 2.0 para projetos Java SE 6 e EE 5. Foi incluído o suporte ao Java Persistence e ao Enterprise Java Beans 3. Além disso, há um novo módulo de suporte Subversion e um módulo atualizado do construtor de GUI, ambos disponíveis na Central de atualizações 5.5.

Destaques do recurso


O NetBeans 5.5 traz muitos recursos que maximizam sua produtividade e flexibilidade para construir aplicativos Java sólidos e em conformidade com o padrão™:
  • Plataforma de suporte geral ao Java Enterprise Edition ™ 5:
    • Implementa o servidor de aplicativos Sun Java System PE 9 (GlassFish).
    • Oferece auto-completar de código e Javadoc para APIs Java EE 5 e descritores de implantação.
    • Tecnologias da camada Web atualizadas para a plataforma Java EE 5: Servlet 2.5, JavaServer™ Pages 2.1, JavaServer™ Faces 1.2
    • Projetos de amostra do Java EE 5 para que você possa se familiarizar rapidamente.

  • Suporte ao Java Persistence:
    • Gera classes de entidade Java a partir de esquemas de banco de dados existentes.
    • Cria classes de entidades no IDE e permite que o servidor gere o banco de dados com base no recurso de banco de dados de Java (DB-from-Java) do GlassFish.
    • O tempo de execução da persistência pode ser usado em aplicativos Web e aplicativos Java comuns sem a necessidade de um servidor de aplicativos Java EE totalmente desenvolvido.
    • Assistentes para o desenvolvimento de um aplicativo da Web completo que utiliza o Java Persistence para proporcionar suporte total a operações de Criação/Leitura/Atualização/Exclusão em um banco de dados.
    • Editor de código-fonte de Java aprimorado, com auto-completar de código, verificações constantes de erros e dicas exclusivas sobre o Java Persistence.
    • Ações de usuário de alto nível que geram padrões de código comuns (por exemplo, usando o Gerenciador de entidades).

  • Suporte à API de Java para serviços da Web XML (JAX-WS), também conhecido como Web Service 2.0.
    • Gera artefatos JAX-WS 2.0 para projetos do SE 6 e EE 5
    • A configuração é feita por meio da personalização de vinculações, tais como chamadas assíncronas e informações do pacote.
    • Assistentes para modelos simplificados e atualizados.
    • Ações de usuário de alto nível que geram padrões de código comuns (por exemplo, chamada a um serviço da Web).

  • Suporte a Beans de sessão do Enterprise Java Beans 3
    • Desenvolvimento simplificado de Beans de sessão e Beans controlados por mensagem.
    • Editor de código-fonte de Java aprimorado, com auto-completar de código, verificações constantes de erros e dicas exclusivas sobre o EJB 3.

  • Outras melhorias:
    • Fácil acesso ao Catálogo de soluções de Java BluePrints do Java EE 5.
    • Auto-completar de código do Idioma de expressão do JavaServer™ Pages .
    • Suporte a clientes de aplicativos Java EE.
    • Suporte à configuração de segurança em aplicativos Java EE.
    • Suporte à execução de scripts SQL diretamente do IDE.

Pacotes suplementares do NetBeans

  • O Enterprise Pack do NetBeans oferece suporte à criação de aplicativos de Arquitetura Orientada a Serviços (SOA), incluindo ferramentas do esquema XML, editor de WSDL e editor de BPEL para orquestração de serviços da Web.
  • Utilize o Mobility Pack para escrever, testar e depurar aplicativos para os dispositivos móveis oferecidos pela tecnologia da plataforma Java Micro Edition (plataforma Java ME).
  • O NetBeans Profiler ajuda você a otimizar a memória de aplicativos e o uso da CPU
  • O C/C++ Native Development Module é compatível com projetos de C/C++.
  • Use o Visual Web Pack para construir rápida e visualmente aplicativos da Web com base nos padrões, incluindo o suporte aos componentes AJAX e JSF.
