什么是 NetBeans？

NetBeans 是一个开源项目，拥有广泛的用户基础和成长中的社区，以及全球100多个（仍不断增加）合作伙伴。Sun Microsystems 于2000年6月创建了 NetBeans 开源项目并且一直以来充当该项目的主要赞助商。

目前有两款产品：NetBeans IDE 和 NetBeans Platform。

NetBeans IDE 是一个开发环境 - 是程序员撰写、编译、测试、调试、性能分析和部署程序的工具。这个 IDE 是用 Java 编写的，但却可以支持任何编程语言。另外还有许多扩展 NetBeans IDE 功能的模块。NetBeans IDE 一个免费的产品，而且没有任何使用限制。

NetBeans Platform 是一个模块化的且可扩展的基础，用作创建大型桌面应用程序的软件基石。将功能丰富的客户端应用程序建立在免费的 NetBeans Platform 基础之上，以便重用 IDE 核心中的功能，从而为应用程序开发节省了大量的时间和精力。ISV 合作伙伴提供增值的插件，它们很容易集成到 IDE 中，并且也可用以开发其本身的工具和解决方案。

两款产品都是开源的，可免费用于商用和非商用目的。源代码在遵循 Common Development and Distribution License (CDDL) 的前提下可以重新再利用。

什么是 netbeans.org？

netbeans.org 是 Netbeans 开源社区的家 - 在这里的人们都致力于建立一个世界级的 IDE。netbeans.org 允许来自全球 160 多个国家的用户连接到 NetBeans 周围的资源和人们。在这里，您可以下载 NetBeans 的最新版本，浏览在线支持文档，强化您的 Java 知识，跟踪最新报道，加入邮件列表，贡献代码，了解涉及同一个项目的人们，建立联系，等等。
