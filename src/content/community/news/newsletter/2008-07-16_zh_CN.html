刊号 # 16 - Jul 16, 2008

项目新闻

已发布: NetBeans IDE 6.1 补丁2

NetBeans IDE 6.1 的一个新的补丁在英语、日语、简体中文和巴西葡萄牙语现已可以获得. NetBeans IDE 必须安装和运行才可以获得该修正. 可以通过 IDE 的插件管理器下载该修正.

NetBeans 可以支持 Python 了 - 加入 NBPython 项目!

在上个星期的 EuroPython 会议中, Sun Microsystems 宣布 NetBeans IDE 将会在将来的版本中支持 Python 和 Jython. NBPython 项目 是由开发者 Allan Davis发起、运行、领导. 参加和对该项目作贡献需要注册为会员. Sun 公司也成立了一个 Python 开发者中心.

文章

NetBeans 6.1 是 InfoWorld's Review of Ruby on Rails IDE 工具中上升最快的!

在 Ruby on Rails 开发中, NetBeans IDE 站在同类产品中前沿! 在 InfoWorld 的最近的九种 Ruby on Rails IDE 工具的对比中, NetBeans IDE 6.1 在十分满分中取得了九分，并被作家 Martin Heller 认为“十分出色”. "如果你已经使用 NetBeans 开发 Java 或者 C/C++ , 那么它肯定也是开发 Rails 的最佳选择. 它是非常好用并高度整合的 Rails IDE 工具，提供了强大的编辑、导航和调试分析功能. 并且它是免费的.”

社区文档组的 EclipseToNetBeans 转移项目

上一年,社区文档组开始了 EclipseToNetBeans 项目,该项目展示了如何由 Eclipse 项目平滑地转移到NetBeans中去. 在本博客入口处,该项目组报导了它的 Eclipse Project Importer 插件已经由更多的功能加强了.

NetBeans 中新的PHP项目向导和更多对PHP支持的升级

NetBeans PHP 项目组欢迎和处理关于 IDE 的 PHP 支持的反馈. 最近在该项目组的博客上的留言改变了创建 PHP 项目的向导、代码完成功能和语法着色功能.
