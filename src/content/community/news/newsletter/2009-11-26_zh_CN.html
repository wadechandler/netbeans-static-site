在 NetBeans IDE 中使用新的NetBeans.org 网站
如你所知, 新的 NetBeans.org 网站是基于 Kenai API 的. 因为 NetBeans IDE 支持 Kenai
API, 你可以使用 NetBeans IDE 中相关的组件. 你只需将 NetBeans.org 作为一个新的 Kenai 实例添加到你的代码并可以学习在我们的服务器上存放的项目代码.
http://services.netbeans.org/newsletter/story.php?id=4458

从 NetBeans 周报邮件列表产生的垃圾信息
上个星期, 由于我们新的发布系统发生了小故障而向订阅了新闻邮件的用户发送一些订阅查询的消息. 我们为本次意外道歉,并采取相关的措施使它不再发生!
http://services.netbeans.org/newsletter/story.php?id=4459

# 文章
--------------------------------------------------------------------------------
在 Netbeans IDE 6.x 中建立 Primefaces 环境
本教程介绍了如何在 Netbeans6.x 中为 JSF 或者带有 Richfaces 的 Facelets 项目建立 Primefaces 组件套件. 你也可以学到怎样在 JSF 中使用 Primefaces 建立一个 Hello World 应用程序. Primefaces 是一个轻量的开源的 JSF 组件, 它包含了 70 多个带有 Ajax 功能的 web 程序的 JSF 组件.
http://services.netbeans.org/newsletter/story.php?id=4452

NetBeans 平台的 Sybase 事务性 SQL 浏览器
SQLBrowser 是一个商业的 Sybase 事务性 SQL IDE 工具, 它是用 NetBeans 平台创建的. 你可以向 SQLBrowser 的开发者 Nicolas Gouteux 学习怎样把程序发布到 NetBeans 平台.
http://services.netbeans.org/newsletter/story.php?id=4453

# 博客
--------------------------------------------------------------------------------
删除 PHP 产生的临时文件
上一周, 在 Petr Pisl 参加 IPC 2009 会议的时候, 一个用户提问了 NetBeans IDE 是否可以删除 PHP 使用的临时空间. Petr 在他的博客中介绍了怎样把这些在不同情况下产生的临时文件(包括版本控制信息文件)删除掉.
http://services.netbeans.org/newsletter/story.php?id=4454

技术贴士: 在 NetBeans 中创建 Grails 项目
Mr. Haki 在他的博客向我们展示了如何在 NetBeans 中使用 Grails 框架创建和配置一个 web 应用程序. 你还可以查看他的其它博文以了解怎样安装 Grails插件, 创建 Grails 文件, 以及如何在 NetBeans 中运行 Grails 命令.
http://services.netbeans.org/newsletter/story.php?id=4455
