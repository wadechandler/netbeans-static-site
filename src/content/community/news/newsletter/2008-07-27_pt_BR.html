Novidades do projeto

Baixe o NetBeans IDE 6.5 Milestone 1!

O NetBeans.org tem orgulho em anunciar o Milestone 1 do NetBeans 6.5. As característica notáveis da versão incluem a integração com funcionaliades do PHP no NetBeans, suporte primário para o Groovy, melhorias nas funcionalidades do Banco de dados e Depurador, e muito mais! Visite nossa página Novo e Notável para maiores detalhes sobre esta e outras características.

A versão final do NetBeans 6.5 está prevista para final de 2008. Nós convidamos você a nos dar seu feedback sobre sua experiência utilizando o NetBeans IDE. Visite nossas listas ou escreva em seu blog!

Video: PHP Editor in NetBeans IDE 6.5

Roman Strobl demonstra as novas funcionalidades disponíveis para o desenvolvimento PHP no NetBeans 6.5 Milestone 1. O Editor PHP na nova versão suporta todas as características padrão como o auto-completar de código, marcação colorida de sintaxe, ocorrência de marcas, refatoração, modelos de código, janela de documentação, navegação de código, avisos do editor e lista de tarefas.

Dicas semanais do NetBeans no NetBeans Zone

Quanto mais atalhos para trabalhar no NetBeans melhor. Novidade no cenário NetBeans: uma série de dicas semanais de Harris Goldstone no NetBeans Zone. Veja e dê seu feedback para Goldstone.

Entrevista com os blogueiros da equipe de desenvolvimento do PHP

A equipe de desenvolvimento do PHP do NetBeans tem tido um duro trabalho implementando as funcionalidades no IDE para os desenvolvedores PHP. Ao longo desta jornada, os membros da equipe iniciaram um blog para documentar os trabalhos. Encontre dois deles: Jan Chalupa e Petr Pisl nesta entrevista discutem o que o NetBeans tem a oferecer para os desenvolvedores PHP.

Artigos

As 10 mais

Zach Elko, membro da comunidade, mostra sua lista de 10 características mais interessantes e úteis no NetBeans. Ele as demonstra com magnificas ilustrações de um projeto C++, embora também possa ser feito com outras linguagens que o NetBeans suporta.

Faça seus Plugins e Aplicativos Compatíveis com o NetBeans Platform 6.1

Quando uma nova versão do NetBeans Platform é liberada, os aplicativos e os plugins construídos em versões anteriores devem funcionar - em teoria. Para ter certeza, os desenvolvedores normalmente verificam se seus produtos ainda são compatíveis com as versões atualizadas da API. No início deste ano, desenvolvedores e empresas parceiras construiram aplicações apenas sobre a plataforma NetBeans. Este artigo apresenta os resultados.

Treinamento

Introdução aos serviços web no NetBeans

Tenha uma visão geral sobre os conceitos do serviços web e as tecnologias suportadas pelo NetBeans. Uma leitura obrigatória para os recém chegados ao mundo dos serviços web antes de entrar nos tutoriais do NetBeans.

JavaMail para iniciantes

Este tutorial de contribuição à comunidade por Angraj Mangal mostra a você como enviar e-mail utilizando Java. Significa que você pode escrever um simples programa Java que envia um e-mail utilizando os detalhes de sua conta de e-mail ou que você pode embutir esta funcionalidade em sua aplicação Web.

Curiosidade: Originalmente um artigo de blog, o tutorial de Mangal foi vencedor no Concurso de Teste de Estudantes da Sun! Parabéns Angraj!

Criando Web Services Axis no NetBeans

Um curto guia para criar e implantar um serviço web Axis2 a partir de uma classe Java no NetBeans. Aprenda a configurar os servidores Tomcat e Glassfish e configurar o IDE para implantar os serviços web Axis.

Conectando equipes de desenvolvimento com o codeBeamer da Intland

Nesta entrevista com Jano Koppany, o CEO da Intland Software, aprenda como o codeBeamer, que está totalmente integrado no NetBeans IDE, está mudando a forma de equipes distribuídas trabalharem juntas.
