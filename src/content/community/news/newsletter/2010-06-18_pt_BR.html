Criar uma Plataforma NetBeans CRUD usando Maven

Este tutorial demonstra como usar o framework de construção Maven para criar um aplicativo da Plataforma NetBeans simples que possa ler e gravar em um banco de dados. Neste documento você irá usar arquétipos do Maven para criar o aplicativo da Plataforma NetBeans e seus módulos, e usará o kit de ferramentas UI Swing e o NetBeans GUI Builder para criar componentes de janelas.

Estados variáveis e paralelos da aplicação no JavaFX Composer

Neste tutorial, você cria uma interface de usuário simples com uma rótulo e dois botões. O aplicativo tem dois conjuntos de estados paralelos. Um conjunto de estados refere-se ao texto do rótulo e outro conjunto de estados diz respeito à cor. Pressione o botão e o texto no rótulo muda. Pressione o outro botão, e a cor do rótulo muda.
