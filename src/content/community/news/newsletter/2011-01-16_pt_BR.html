Agenda

Webinar gratuito ao vivo: Desenvolvimento de Aplicações Web com Java EE 6, GlassFish e NetBeans (20 de janeiro)

Junte-se ao evangelista Oracle Arun Gupta para um webinar de código intensivo que demonstra a facilidade de desenvolvimento com Java EE 6, GlassFish e NetBeans. Descubra como as três tecnologias podem ser usadas em conjunto para construir aplicações web e serviços REST. Cadastre-se agora.

Horário: 10:00 PDT / 1:00 pm EDT / 19.00 CET (Verifique seu fuso horário)

NetBeans, JDK 7 e Java EE 6 no JFokus, Suécia (14-16 fevereiro)

Junte-se aos evangelistas Oracle Arun Gupta, Alexis Moussine-Pouchkine, Roger Brinkley e Tomas Nilsson na JFokus em 2011, em Estocolmo, Suécia, para aprender sobre os mais recentes recursos em Java EE, GlassFish e JDK 7. Como o NetBeans IDE em conjunto com estas tecnologias ajuda os desenvolvedores a serem mais produtivo, e muito mais.

Novidades do projeto

Plugin NetBeans: PDF Debugger

RUPS (Reading and Updating PDF Syntax - “Leitura e Atualização da sintaxe PDF”) é uma ferramenta criada pelo iText para depurar documentos PDF. Leia mais sobre o plugin do NetBeans aqui.

SevenBeans: Integração NetBeans com Windows 7

SevenBeans é um módulo do NetBeans que tornam o IDE, e outros aplicativos da plataforma NetBeans, mais produtivo no Windows 7. Agora você pode fixar o NetBeans na barra de tarefas, ter acesso dos projectos mais recentes de forma mais rápida, ver o progresso e status diretamente no botão da barra de tarefas.
