NetBeans 6.1 est élu meilleur EDI pour Ruby on Rails par le magazine InfoWorld !

Dès qu'il a abordé le développement de Ruby on Rails, l'EDI Netbeans a pris la tête du classement ! Dans le récent comparatif de la revue InfoWorld de neuf EDIs de Ruby on Rails, Netbeans 6.1 a eu 9.0 sur 10 et a reçu la mention "Excellent" par l'écrivain Martin Heller.

" Si vous utilisez déjà Netbeans pour le développement Java et/ou C/C++, alors il doit être probablement au top de votre choix comme EDI pour Rails. Fournissant une solide édition et navigation avec un excellent deboggeur et profilage, il est vraiment capable et intégre complètement IDE pour Rails. Et il est gratuit.”

Nouvelles de projet

Sortie: Patch2 pour NetBeans 6.1

Un nouveau patch est disponible pour Netbeans 6.1 en Anglais, Japonais, Chinois et Portugais brésilien. Pour obtenir les correctifs, l'EDI Netbeans doit être installé et lancé. Télécharger les correctifs à travers le gestionnaire de plugins de l'EDI.

Netbeans supporte Python - Rejoindre le projet NBPython !

La semaine passée à la conférence EuroPython , Sun Microsystems a annoncé que Netbeans va supporter Python et Jython dans les futures versions. Le Projet NBPython a été crée, est fonctionnel et dirigé par le développeur Allan Davis. Démander une adhésion pour rejoindre le projet ou apporter des contributions. Sun a aussi lancé un centre pour développeur Python.

Contruire des rélations entre les modèles de Rails

Le framework Ruby on Rails a un puissant sous-framework de modèle connu sous le nom de Active Record. Ce dernier analyse les schémas de la base de données et fournit de façon automatique la plupart des méthodes nécessaires pour manipuler les données. Apprenez à utiliser les méthodes de Active Record pour spécifier la rélation un-à-plusieurs entre deux modèles de classes.
