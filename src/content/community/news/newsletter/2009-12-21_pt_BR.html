Veja já hoje o amanhã com o NetBeans IDE 6.8!

O NetBeans IDE 6.8 está disponível para download! Esta versão oferece o melhor em suporte para toda a especificação Java EE 6 e à plataforma GlassFish Enterprise Server v3. Incluindo destaques adicionais para novas funcionalidades e maior suporte para PHP 5.3 e framework symfony, o mais novo JavaFX SDK 1.2.1, C/C++, Ruby, Maven e muito mais. Como sempre, contamos com a plataforma NetBeans para uma estrutura de aplicação sólida que poupa muitos anos de desenvolvimento. Baixe a nova versão e descubra porque o NetBeans continua a ser a escolha dos desenvolvedores de qualquer todo o mundo!

    • Versão prévia: Plugin Compositor JavaFX

    O Compositor JavaFX é uma ferramenta de layout visual para criar aplicativos do JavaFX. Com a ferramenta, os desenvolvedores JavaFX podem criar GUIs com controles padrão JavaFX, utiliza estados para definir o fluxo da aplicação, e muito mais. O Compositor JavaFX é uma versão de pré-lançamento e está disponível para o NetBeans IDE 6.8 como um plugin no NetBeans Update Center. (Dentro do IDE NetBeans, vá em Ferramentas -> Plugins -> Plugins Disponíveis -> JavaFX.) Para saber mais sobre o compositor e tutoriais, visite o página wiki do JavaFX Composer.
