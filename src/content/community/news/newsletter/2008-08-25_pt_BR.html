Plugin OpenLaszlo para o NetBeans disponível

OpenLaszlo é um framework de aplicações ricas para a internet (RIA). Baseado em Flash, permite aos desenvolvedores construirem aplicações ricas (RIA) usando XML e JavaScript sem conhecimentos de Flash. O plugin OpenLaszlo para o NetBeans é composto pela biblioteca OpenLaszlo 4.1.1 e inclui algumas aplicações de exemplo, modelos, suporte para arquivos .lzx e muito mais. O plugin funciona com NetBeans 5.5.1, 6.0 e 6.1.

SaaS (Software as a Service): Tutorial Zillow

NetBeans IDE 6.1 fornece um Gerenciador de Serviços Web que suporta aplicativos de Software como um Serviço (SaaS). Zillow é uma empresa de serviços imobiliários online de Washington que permite aos usuários visualizarem o valor de milhoes de lares ao redor dos Estados Unidos.
