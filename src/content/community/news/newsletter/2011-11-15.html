Project News

Plugin: NetBeans Cubeon Plugin

Active development is back on for the NetBeans Cubeon plugin project. Check out version 1.2.0.0, the latest release.

NBAndroid Plugin Updated

An update to the NBAndroid plugin is now available. The release now supports the current version of Android SDK tools, enhancing your work when developing Android applications in the NetBeans IDE.
