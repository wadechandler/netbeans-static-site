Call for Nominations: Oracle's Java Innovation 2011 Awards

Do you know of a developer, a team or an organization accomplishing creative and innovative implementation with Java technology? Nominate them for the 2011 Oracle Excellence Awards!

Winners will receive a Duke Statue, a free JavaOne full conference pass, and their winning implementation will be showcased at JavaOne. Hurry! The deadline for nominations is Friday, July 29th.

Contribute Your Java Expertise to Oracle's Java 7 Summit

Oracle's Java 7 Summit (November 2-4) offers Java developers and architects an in-depth technical education on the recent Java 7 release and on OpenJDK. Co-located with EclipseCon Europe 2011, the Java Summit will feature expert speakers from Oracle and the wider OpenJDK community. Java experts are invited to propose sessions for the Summit. The deadline for submissions is August 17.
