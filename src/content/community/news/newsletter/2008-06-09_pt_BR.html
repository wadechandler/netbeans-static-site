Introdução ao desenvolvimento SunSpot utilizando o NetBeans

Um "how-to" iniciante para construir uma simples demo de aplicativo Sun SPOT no NetBeans. Sun SPOT (Sun Small Programmable Object Technology) é um sensor de rede sem-fios (um dispositivo de comunicação eletrônica com o intuito de ser do tamanho de uma partícula de poeira) desenvolvido pela Sun Microsystems.

Módulo de modo de editores somente-leitura

Controle a propriedade editável dos editores de texto com o módulo de modo de editores somente-leitura de Sandip Chitale. O estado inicial da propriedade editável do editor de texto é controlado por: Ferramentas:Opções:Micelâneas:Modo de editores somente leitura. O usuário pode alternar o modo somente-leitura utilizando a ação da barra de ferramenta do editor (botão com ícone de um cadeado aberto/fechado). A chave de ligação padrão para alternar esta ação é o CTRL + R. Nota: Muitos dos módulos Sandip estão agora disponíveis no NetBeans Plugin Portal.
