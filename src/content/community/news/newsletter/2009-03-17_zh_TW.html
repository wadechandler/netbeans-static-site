專案新聞

NetBeans IDE 6.5.1 現在登場

NetBeans IDE 6.5.1 是 NetBeans 6.5 的微幅改版。 此次釋出版本包含了英文， 簡體中文， 日文， 以及巴西版葡萄牙文， 且包含了下面幾個修改: * GlassFish v2 UR2 由 GlassFish v2.1 所取代 * 整合 2008 年十二月至 2009 年一月的所有更新修正 * 整合 JDK6 Update 12 的錯誤修正及其他問題 請參訪 NetBeans IDE 6.5.1 資訊頁面來取得更多詳細資訊， 以及今天就立刻下載 NetBeans 6.5.1 吧。

BlueJ 以及 Greenfoot 開放程式碼

BlueJ 團隊已經宣佈旗下的 BlueJ 以及 Greenfoot 系統將開放其程式碼。兩者皆以與 NetBeans IDE 所使用相同的 Classpath Exception 為但書下的 GNU 通用公共授權第二版之下釋出其程式碼。 BlueJ 是個給大學生練習程式設計與 Java 很好的一種方式。 從 BlueJ "畢業"之後， 他們可以繼續使用 NetBeans BlueJ 外掛 來轉移至 NetBeans IDE。 Greenfoot 是個用來對中學(及更年輕的)學生進行 Java 以及程式設計的一個工具。
