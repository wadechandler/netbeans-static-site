JavaFX 2.0 Beta 版的 NetBeans 插件已经可以下来了

万众瞩目的 JavaFX 2.0 已经开始公开测试了，这意味着你可以使用所有 JavaFX 2.0 带给 Java 平台的新特性。这是 Oracle 承诺将 JavaFX 打造成最好的富客户端平台后的最新开发版本。
JavaFX 2.0 beta 版 NetBeans IDE 7.0 插件可以通过 javafx.com 获得。请通过测试和反馈你遇到的问题来推荐 JavaFX。浏览最新 API 和文档。
