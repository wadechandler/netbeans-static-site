Novidades do projeto

Baixe o NetBeans IDE 6.9 Milestone 1

O NetBeans IDE 6.9 Milestone 1 está disponível para download. Os destaques desta versão de desenvolvimento estabilizada inclui novos recursos e melhorias para a interoperabilidade do NetBeans Platform OSGi, editor Java, Java EE, PHP, C/C++ e muito mais.

Visite a página de Notas e Novidades do NetBeans 6.9 para saber mais sobre todos os recursos disponíveis neste milestone. A versão final do NetBeans IDE 6.9 está previsto para o inverno de 2010.

Novas ferramentas arrojadas para os desenvolvedores OSGi (DZone)

A equipe do NetBeans está trabalhando em novas ferramentas para desenvolvedores OSGi, especificamente para aqueles que querem usar o Swing como toolkit de interface gráfica. A Plataforma NetBeans irá reconhecer os pacotes OSGi, e como resultado os usuários serão capazes de importar pacotes OSGi em seus aplicativos da plataforma NetBeans.

Formatação de linhas brancas no Editor PHP

Na versão de desenvolvimento do NetBeans IDE 6.9, há uma nova categoria de formatação para PHP: a categoria de linhas brancas. Você pode definir onde quer a formatação do código para adicionar ou excluir linhas em branco entre os diferentes elementos no código PHP.
