Novidades do projeto

NetBeans 6.5 Beta está disponível

O NetBeans 6.5 Beta traz muitas novidades, incluindo um IDE robusto para PHP, depurador JavaScript para Firefox e IE, e suporte para Groovy e Grails. Esta liberação também inclui muitas melhorias para o Java, Ruby e Rails, e desenvolvimento C/C++. Os destaques para o Java incluem: suporte para funções internas do Hibernate, importação do projeto Eclipse, e compilar ao salvar.


Software NetBeans em seu idioma

Esta semana, o projeto translatedfiles do NetBeans iniciou a tradução do IDE 6.5. Junte-se ao projeto se você estiver interessado em traduzir o NetBeans Platform em seu idioma!

Portal Pack 2.0.1 ML está disponível

Portal Pack 2.0.1 está disponível para download. Os destaques incluem:

  • Suporte ao Public Render Parameter no Eventing Story Board
  • JSF portlet bridge 1.2.3 library
  • Correção de bugs
  • Suporte aos idiomas inglês, japonês, chinês e português.

O plugin também pode ser baixado pela Central de Atualização Automática do NetBeans 6.1.

Comunidade

Baixe uma licença gratuíta para desenvolvedores do ChartFX 7

O Chart FX 7 para Java não deixa rastros, graças a sua UI totalmente interativa à implementação de um framework independente de AJAX. Combinando Chart FX e NetBeans, os desenvolvedores estão prontos para utilizar as mais de 500 propriedades e características dinâmicas, desde menus de contexto e barras de ferramentas a paineis de negócios avançados que combinam gráficos, medidores e mapas para uma experiência rica e interativa.

NOVO! Fórum de suporte do NetBeans

Web ou e-mail? Você escolhe! O NetBeans.org anuncia o novo Fórum da comunidade NetBeans. Este fórum é sincronizado com a maioria dos usuários das listas de discussões do NetBeans: isto significa por exemplo que, um comentário enviado ao fórum dos "Usuários do NetBeans" também irá aparecer na lista de discussão nbusers, e vice-versa. Comunique-se!

Criando uma aplicação Ajax com Script.aculo.us

Aprenda como criar, recuperar, atualizar, e excluir linhas do banco de dados com o NetBeans 6 e Script.aculo.us, um conjunto de bibliotecas JavaScript para melhorar a interface de usuário de sites web.

Tutorial da API do Auto-completar do Editor

Neste tutorial Geertjan explica como os desenvolvedores do NetBeans Platform podem implementar o suporte MIME-specific para o auto-completar de código em suas próprias aplicações.

Lightweight UI Toolkit: Desenvolvimento Desktop para Java ME

LWUIT é uma biblioteca avançada de Java ME para desenvolver interfaces poderosas que anulam as restrições lcdui de formulários e não reinventa a roda. Nesta série de artigos, Ibon Urrutia mostra como desenvolver um simples, mas completo, aplicativo demonstrando as diferentes capacidades desta biblioteca.

Iniciando no OpenLaszlo e NetBeans

OpenLaszlo é um framework web, baseado em flash, de aplicações ricas para internet (RIA - Rich Internet Application). O vídeo de Sotohiro Terashima mostra como instalar o Plugin OpenLaszlo e criar uma aplicação web baseada no OpenLaszlo.
