项目新闻

JavaFX SDK 1.1 现已发布，并可在 NetBeans IDE 6.5 中使用！

JavaFX SDK 1.1 将 JavaFX 应用程序带向了移动设备。SDK 现在支持 JavaFX 移动版，并包括一个移动仿真器。NetBeans IDE 是 JavaFX 的推荐开发环境。今天就下载用于 JavaFX 的 NetBeans IDE，实际体验 JavaFX 移动版！

（如果你已经安装了 NetBeans IDE 6.5，可使用 IDE 的“更新中心”来添加 JavaFX 1.1。在 IDE 中，导航到“工具”->“插件”，然后选中 JavaFX 类别下的复选框。）

欲知更多信息：

  • NetBeans IDE for JavaFX - 功能
  • 用于 JavaFX 的NetBeans IDE - 文档
  • 访问 JavaFX.com 以获得教程和样例应用程序


Portal Pack 3.0 现已可下载

有着众多的新功能和错误修复，Portal Pack 3.0 使 portlet 开发比以前更容易。本次发布同样和 NetBeans 6.5 IDE 兼容。从这里了解更多，并可查看这份关于在 NetBeans 中使用 Portal Pack 3.0 插件的快速入门指南。
