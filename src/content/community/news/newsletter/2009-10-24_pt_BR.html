NetBeans IDE 6.8 Beta disponível para Download

A equipe do NetBeans tem o prazer de anunciar o NetBeans IDE 6.8 Beta

  • Download NetBeans IDE 6.8 Beta
  • Saiba mais sobre o NetBeans IDE 6.8 Beta
O NetBeans IDE 6.8 Beta é o primeiro a oferecer suporte completo para o Java EE 6 spec. Os destaques incluem suporte para JSF 2.0/Facelets, Java Persistence 2.0, EJB 3.1 incluindo utilização de EJBs em aplicações Web, serviços Web RESTful, e GlassFish v3. A integração do IDE com o Project Kenai, um ambiente colaborativo para hospedar projetos open-source, agora oferece suporte completo para JIRA e melhoria ao instant messenger e integração ao issue tracker. O suporte ao PHP foi estendido para incluir o framework Symfony e PHP 5.3. O lançamento também suporta o JavaFX SDK 1.2.1, e vem com características adicionais para Maven e integração à banco de dados, além de melhorias para o editor e ferramentas para Ruby, Groovy, e C/C++. O NetBeans IDE 6.8 Beta está disponível em Inglês, Português brasileiro, japonês e chinês simplificado, bem como em várias línguas traduzidas pela comunidade. Destaques da Versão
  • Java Enterprise Edition 6
    • JavaServer Faces 2.0 para interfaces web e a capacidade de usar EJBs em aplicações web
    • Java Persistence JPA 2.0 e suporte a serviços Web RESTful
    • Ampliado o uso de anotações em vez de descritores de implantação
    • Implantação, depuração e profiling com GlassFish v3
  • JavaServer Faces 2.0 (Facelets)
    • Auto-completar de código, sugestões sobre erros, auto-completar para namespaces, janelas de documentação, e auto-importação de tag para Facelets
    • Suporte às bibliotecas Facelets, componentes compostos, linguagem de expressão
  • JavaFX
    • Suporte para o JavaFX SDK 1.2.1
    • Navegação melhorada, auto-completar de código, e dicas
  • PHP
    • Suporte completo ao PHP 5.3
    • Suporte ao Framework Symfony
  • Kenai.com: Desenvolvedor Conectado
    • Suporte completo ao JIRA
    • Melhoria à integração de issue tracker
  • Maven
    • Melhoria ao suporte para Java EE 6, Groovy, Projetos Scala
    • Exclusão de dependência personalizável em gráfico de dependência
  • Ruby
    • Suporte para o JRuby 1.3.1, depuração Ruby 1.9 e RSpec 1.2.7
    • Melhoria ao renomear da refatoração, inferência de tipos e navegação
  • C / C + +
    • Profiling: Novas ferramentas para a Monitoria de E/S, análise de thread e detecção de disputa.
    • Sincronização mais rápida durante o desenvolvimento remoto

Plugin ALM com suporte para Mercurial e Git

Um grupo de entusiastas do software open source e a Intland continuam a desenvolver o plugin NetBeans ALM (Application Lifecycle Management). Este plugin se integra com o codeBeamer 5.4, Mercurial e Git. A equipe solicita feedback de todos, e incentiva os membros da comunidade NetBeans para aderirem ao projeto no JavaForge. O lançamento está agendado para 31 de janeiro de 2010.
