專案新聞

NetBeans 不落人後，JavaFX SDK 1.1 in NetBeans 釋出囉！

藉由 JavaFX SDK 1.1 ，JavaFX 已可跨足行動裝置。該 SDK 現已支援 JavaFX Mobile 並且包含行動模擬器( Mobile Emulator )。而 NetBeans 更是開發 JavaFX 強力推薦的 IDE。趕快下載支援 JavaFX 的 NetBeans 版本吧！心動不如馬上行動。 (若您已安裝 NetBeans 6.5，則可以直接從更新中心取得 JavaFX 1.1。請至 NetBeans 的 工具->外掛程式 並選取 JavaFX 分類之下的所有方框。) 想知道更多：

  • 支援JavaFX 的 NetBeans 版本 - 特色
  • 支援JavaFX 的 NetBeans 版本 - 說明文件
  • 至 JavaFX.com 取得教學文件與範例程式


Portal Pack 3.0 出爐囉！

一如往常地，少不了新增特色與錯誤修正。Portal Pack 3.0 也讓 portlet 開發更簡單了，同時在可以在 NetBeans 6.5 之上使用。想知道更多請按此或觀看NetBeans 的 Portal Pack 3.0 外掛使用簡介。

3 個讓您不得不把 JavaFX Mobile 加進 JavaME 行動程式的理由

Bruce Hopkins 用三個例子來向諸位開發行動裝置程式的開發者說明 JavaFX 優點。開發人員可以呼叫原本在 Java ME 與 Java SE 程式庫裡的物件方法並實體化其中的物件 – 即在 JSR 82、135、180 的API中的物件。此外 JavaFX 還可以把一個物件的值連結到另一個變數呢！

部落格串文 – NetBeans Lookup Library

Antonio Viero 有四篇文章介紹 NetBeans Lookup Library (譯者注：在一個component-based system (基於元件的系統)各個不同元件會向系統註冊，但為了解決各元件之間要如何查詢其他元件的問題，故而開發 Lookup Library以解決此問題。)。得知更多如何加入 NetBeans Lookups、建立 Lookup 物件、使用NetBeans Lookup API 加入功能於既有物件之中、以及在執行期間偵測 Lookup 物件的變更。 若此篇電子報翻譯有任何需要改進之處，歡迎email至 hwjnctucsie92@gmail.com。
