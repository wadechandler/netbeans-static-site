Télécharger une licence gratuite de ChartFX 7 Developer jusqu'au 31 août!

Portal Pack 2.0.1 ML est maintenant disponible

Portal Pack 2.0.1 est maintenant disponible pour téléchargement. Les caractéristiques de cette version comprennent :

  • Le support pour le paramétrage de rendu public dans un inventaire de story board
  • une librairie pour JSF portlet bridge 1.2.3
  • les bugs corrigés
  • un support pour l'anglais, japonnais, le chinois simplifié et le portugais du Brésil.
Le plug-in peut aussi être téléchargé automatiquement à partir du centre de mise à jour pour Netbeans 6.1.

NetBeans IDE 6.5 Beta est maintenant disponible

NetBeans IDE 6.5 Beta présente de nombreuses fonctionnalités dont un robuste EDI pour PHP, le débogage de JavaScript pour Firefox et IE et un support pour Groovy et Grails. Cette version inclut également un certain nombre d'améliorations pour Java, Ruby and Rails et le développement C/C++. Les mises en valeurs des fonctionnalités Java incluent : un support intégré pour Hibernate, l'import de projet Eclipse , compiler et enregistrer.
