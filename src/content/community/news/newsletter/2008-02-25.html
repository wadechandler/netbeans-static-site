Project News

NetBeans Innovators Grants - Submit Your Proposals by March 3!

Time is running out to submit those fantastic project ideas for the NetBeans Innovators Grants program. The deadline is March 3! Your project can be a module, an application on top of the NetBeans platform, or even non-code related (books or translations). Stumped about what to work on? Browse our list of project ideas!

Listen to Community Manager Bruno Souza discuss the grant program on Sys-con TV.

NetBeans 6.0.1 Bug Fixes Now Available

NetBeans IDE 6.0.1 Patch1 is a small update to the NetBeans IDE 6.0. The update includes several bug fixes in modules for: Mercurial SCM Plugin, GlassFish Integration, IDE Platform, Java, Java EE, Java Persistence, Java Profiler, JSF, Mobility, Ruby and Rails, Struts, UML, Visual JSF, and Web Applications.

To obtain the fixes, the NetBeans IDE 6.0 must be installed and running. You can download the fixes through the IDE's Plugin Manager.
