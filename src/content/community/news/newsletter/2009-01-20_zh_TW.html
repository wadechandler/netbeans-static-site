NetBeans 喜獲 Developer.com 2009 作品大獎

狂賀 NetBeans 在 2009 開春好運到！NetBeans 一舉嬴得了 Developer.com 2009 作品大獎中數個獎項。 NetBeans 經票選過後獲得以下五個獎項：最佳 Java 工具、最佳開放源始碼工具、最佳開發平台、最佳開發效能、以及最佳無線/行動工具。 在此特別感謝 NetBeans 社群及使用者們的支持以使 NetBeans 榮獲此獎。


Portal Pack 3.0 Beta 已正式發行！

Portal Pack 3.0 Beta 已正式開放下載。此次釋出包含了主要錯誤修正，以及增進資料夾(路徑)配置與支援 Liferay Portal Server 於 Tomcat 執行時的除錯功能。此外尚增加了以下的支援功能：

  • WebSynergy 專案
  • Liferay Portal Server 5.1.x 以上之版本
  • Sun Java System Portal Server 7.1/7.2
  • OpenPortal Portlet 容器( Container )
