NetBeans partners are companies that provide, or help provide, a plugin or service for NetBeans IDE to support popular features, frameworks, technologies, and platforms. NetBeans partners provide a value add for NetBeans developers with a business model of their choosing, which could be for a fee or for free.

This page highlights NetBeans partner offerings to the NetBeans community, while the complete listing of NetBeans plugins is available from the NetBeans Plugin Portal.

Alice 3

Alice 3 is a 3D programming environment, freely available, designed to be a student's first exposure to object-oriented programming in Java. Alice allows students to learn fundamental programming concepts in the context of creating animated movies and simple video games. In Alice, 3-D objects (e.g., people, animals, and vehicles) populate a virtual world and students create a program to animate the objects using a drag-and-drop interface.

Alice 3 provides a NetBeans plugin that allows students to import the project they started in the Alice environment into NetBeans IDE, continuing their project as they learn and expand their understanding of Java.

TestFairy

TestFairy helps Android developers do painless super smart app testing. When testing Android apps, whether it is your co-workers, social media friends, or paid testers, you never know what exactly was tested and what exactly went wrong on the client side. TestFairy solves this problem by showing you a video of the exact test that was done, including detailed internal metrics such as CPU, Memory, GPS, Network, phone reception, battery usage and a lot more. The service is extremely easy to use, there is no SDK and no code to add to the tested app, all changes are made in real time on the TestFairy platform.

The NetBeans TestFairy plugin provides a "Send to TestFairy" action on APK files. When clicked, the APK file is sent to TestFairy and a moment later the instrumented APK file is returned from TestFairy, ready to be distributed to your testers. Then you can monitor what is being tested by going to the page dedicated to your APK file on TestFairy.com.
