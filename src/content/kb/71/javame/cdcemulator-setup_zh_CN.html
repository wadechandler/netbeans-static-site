NetBeans CDC 仿真器平台安装指南

尽管 NetBeans 支持 Java Micro Edition (ME) 联网的设备配置 (Connected Device Configuration, CDC) 开发，但是 IDE 并未在安装程序中包含 CDC 平台，您可以通过此平台为使用 Personal Profile、Foundation Profile 或 AGUI Profile 的 CDC 设备创建应用程序。本文档将提供有关对 Sun Java Toolkit for CDC 以及其他用于 CDC 开发的常用软件开发工具包 (Software Development Kit, SDK) 进行下载、安装和注册的信息。

所需的软件

要学完本教程，您需要具备以下软件和资源：

软件或资源 要求的版本
带有 Java ME 的 NetBeans Beans 版本 6.9 或更高版本
Java Development Kit (JDK) 版本 6 或版本 7

安装仿真器平台

Nokia Series 80 Platform SDK for Symbian OS, for Java, Personal Profile

Nokia 80 系列设备的开发者可以按照以下说明安装 Series 80 Platform SDK for Symbian OS, for Java, Personal Profile。

  1. 从 Nokia 开发者论坛下载 Nokia Series 80 Platform SDK for Symbian OS, for Java, Personal Profile（下载）。文件名为 S80_DP_2_0_PP_SDK.zip。
  2. 解压缩该文件。
  3. 导航至该文件在系统上的解压缩位置。双击 setup.exe 文件图标。将打开 Nokia 安装程序向导。
  4. 按照向导说明安装 SDK。该向导将提示您设置缺省设备。选择 "Series80_DP2_0_PP_SDK"。
  5. 从 NetBeans IDE 的主菜单中，选择 "Tools"（工具）> "Java Platforms"（Java 平台）。
  6. 单击 "Add Platform"（添加平台）。“添加 Java 平台”对话框打开。
  7. 选择 "Java ME CDC Platform Emulator"（Java ME CDC 平台仿真器），然后单击 "Next"（下一步）。
  8. 选择包含平台 SDK 的文件夹。缺省目录为 C:\Symbian\S80_DP2_0_PP_SDK。单击 "Next"（下一步）。
  9. 如果需要，请使用 "Platform Name"（平台名称）对话框来添加或删除额外的资源库。
  10. 单击 "Finish"（完成）。IDE 将注册 CDC 平台。

CrEme VM for Windows CE

Windows CE 设备的开发者可以按照以下说明安装 NSIcom CrEme VM for Windows CE：

  1. 从 NSIcom 下载 CrEme 4.12 平台（下载）。
  2. 导航至 CrEmeDevSup410.exe 文件在系统上的下载位置。双击该文件。
  3. 按照 NSI.com InstallShield 向导的说明安装开发者工具包。
  4. 从 NetBeans IDE 的主菜单中，选择 "Tools"（工具）> "Java Platforms"（Java 平台）。
  5. 单击 "Add Platform"（添加平台）。“添加 Java 平台”对话框打开。
  6. 选择 "Java ME CDC Platform Emulator"（Java ME CDC 平台仿真器），然后单击 "Next"（下一步）。
  7. 选择包含平台 SDK 的文件夹。缺省目录为 C:\Program Files\NSIcom\CrE-ME V4.12。单击 "Next"（下一步）。
  8. 如果需要，请使用“平台名称”对话框添加或删除其他资源库。
  9. 单击 "Finish"（完成）。IDE 将注册 CDC 平台。

注意：JFrame 和 GroupLayout 不受 CrEme 评估许可证的支持，但 NSI.com 提供获取高级图形支持的下载页。
