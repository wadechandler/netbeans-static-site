Vídeo da Criação de uma IU SVG Ativada por Toque para Dispositivos Java ME

Este screencast demonstra como desenvolver uma aplicação que funciona na maioria dos dispositivos de tela sensível ao toque Java ME usando widgets de interface do usuário com SVG (Scalable Vector Graphics) no NetBeans IDE.

Esta demonstração é baseada no tutorial Criação de uma IU SVG Ativada por Toque para Dispositivos Java ME que descreve todos os pré-requisitos necessários para criar a aplicação em foco.
