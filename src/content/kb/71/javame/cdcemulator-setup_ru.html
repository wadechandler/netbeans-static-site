Руководство по установке платформы эмулятора CDC в NetBeans

Несмотря на то, что NetBeans поддерживает разработку для Connected Device Configuration (CDC) на платформе Java Micro Edition (ME), программа установки среды IDE не содержит платформы CDC, которая позволила бы создавать приложения, поддерживающие устройства, в которых используются профили Personal Profile, Foundation Profile или AGUI. В этом документе приводятся сведения о загрузке, установке и регистрации инструментария Sun Java Toolkit для CDC и других популярных пакетов (SDK) для разработки CDC.

Установка платформ эмулятора

Пакет SDK платформы Nokia Series 80 для ОС Symbian, Java, Personal Profile

Для разработчиков на устройствах Nokia 80 Series процедура установки пакета SDK платформы Series 80 для ОС Symbian, Java, Personal Profile заключается в следующем.

  1. Загрузите пакет SDK платформы Nokia Series 80 для ОС Symbian, Java, Personal Profile с форума разработчиков Nokia (загрузить). Имя файла – S80_DP_2_0_PP_SDK.zip.
  2. Распакуйте файл.
  3. Перейдите к каталогу, куда был распакован файл. Дважды щелкните значок файла setup.exe. Откроется мастер установки Nokia.
  4. Для установки пакета SDK следуйте указаниям мастера. Мастер выведет запрос на выбор устройства, которое будет использоваться в качестве устройства по умолчанию. Выберите устройство Series80_DP2_0_PP_SDK.
  5. В главном меню среды IDE NetBeans выберите "Tools" > "Java Platforms".
  6. Нажмите кнопку "Add Platform". Откроется диалоговое окно "Add Java Platform".
  7. Выберите "Java ME CDC Platform Emulator" и нажмите кнопку "Next".
  8. Выберите папку, содержащую пакет SDK платформы. Каталог установки по умолчанию – C:\Symbian\S80_DP2_0_PP_SDK. Нажмите кнопку "Next".
  9. При необходимости добавьте или удалите дополнительные библиотеки ресурсов с помощью диалогового окна "Platform Name".
  10. Нажмите кнопку "Finish". Платформа CDC будет зарегистрирована в среде IDE.

Виртуальная машина CrEme для Windows CE

Для разработчиков на устройствах Windows CE процедура установки VM CrEme NSIcom для Windows CE заключается в следующем:

  1. Загрузите платформу CrEme 4.12 из NSIcom (загрузить).
  2. Перейдите к каталогу, куда был загружен файл CrEmeDevSup410.exe. Дважды щелкните файл.
  3. Следуйте указаниям мастера "NSI.com InstallShield" для установки комплекта для разработчика.
  4. В главном меню среды IDE NetBeans выберите "Tools" > "Java Platforms".
  5. Нажмите кнопку "Add Platform". Откроется диалоговое окно "Add Java Platform".
  6. Выберите "Java ME CDC Platform Emulator" и нажмите кнопку "Next".
  7. Выберите папку, содержащую пакет SDK платформы. По умолчанию используется каталог C:\Program Files\NSIcom\CrE-ME V4.12. Нажмите кнопку "Далее".
  8. При необходимости добавьте или удалите дополнительные библиотеки ресурсов с помощью диалогового окна "Platform Name".
  9. Нажмите кнопку "Готово". Платформа CDC будет зарегистрирована в среде IDE.

Примечание. JFrame и GroupLayout не поддерживаются оценочной лицензией CrEme, однако NSI.com предоставляет возможность загрузки для дополнительной поддержки графики.
