Создание документации PHP в среде IDE NetBeans

В этой экранной демонстрации показано, как создавать документацию PHP с помощью средства PHPDocumentor в среде IDE NetBeans. Теперь в среду IDE NetBeans 7.0 добавлена встроенная поддержка PHPDocumentor. В этой экранной демонстрации содержатся подробные сведения по установке PHPDocumentor с помощью команды PEAR, а также по настройке PHPDocumentor, среды IDE NetBeans и файла php.ini. Это видео будет особенно полезно разработчикам, использующим среду NetBeans в ОС Microsoft Windows.

Внимание! PHPDocumentor не предоставляет полной поддержки PHP 5.3. Пространство имён и ряд других функций выпуска 5.3 не документируются средством PHPDocumentor.
