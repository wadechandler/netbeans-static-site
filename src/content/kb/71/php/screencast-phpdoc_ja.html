NetBeans IDE での PHP ドキュメントの生成

このスクリーンキャストでは、NetBeans IDE で PHPDocumentor を使用して、PHP ドキュメントを生成する方法を説明します。PHPDocumentor の組み込みサポートは NetBeans IDE 7.0 で追加されました。このスクリーンキャストには、PEAR を使用した PHPDocumentor のインストールと、PHPDocumentor、NetBeans IDE、および php.ini ファイルの構成についての詳細が含まれています。このビデオは、NetBeans を Microsoft Windows オペレーティングシステム上で使用する開発者にとって特に役立ちます。

警告: PHPDocumentor では、PHP 5.3 は完全にはサポートされていません。名前空間およびその他いくつかの 5.3 機能は、PHP Documentor ではドキュメント化されません。
