Создание мобильных игр в IDE NetBeans

В этой экранной демонстрации рассматриваются способы создания игры Java ME с использованием NetBeans Mobile Game Builder, а также использование Game Designer для создания отображаемых элементов игры, например персонажей и плиточных слоев. В основе этой демонстрации лежит практическая лабораторная работа Лукаса Хасика (Lukas Hasik), представленная на конференции JavaOne.
